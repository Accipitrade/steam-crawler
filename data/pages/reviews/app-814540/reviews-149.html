\r\n\t\t\t\t\tThis is by far one of the best games art-wise I have ever seen and played. DragonSnow, the artist for this game did a fanominal job at creating the animation that play out though out the game and the various unique game over screens you may encounter throughout the game. The game play, I will describe as decent. The player controls well and the game only gives you 10 options to choose from at any given point: walk up, walk right, walk down, walk left, run up, run right, run down, run left, literally do nothing and wait, interact with the object you are facing. The game isn't too difficult, it took me about 6 hours to complete the game on normal mode. There is no hard mode, but there is an easy mode. And I believe it took me about another 2 hours-ish to get all the other achievements. The mechanics of the game are very easy to understand and getting a game over will not set you back far due to the numerous save point though out the game. The puzzles are not easy but they do give the player a good challenge occupationally. Now part of the appeal of this game is the actual game over animations and there are secret ways to get unique game overs that you need to go out of your way to find. Again, due to how well done the game overs are done in this game, getting these game over screens are worth it for most players in my opinion. With this said, I did not find this game to be that sexualized, however there are some animations that do look like something else mostly due to animation size limitations. With that said, the concept of vore via goo creature and transformation are the only fetishes I know about and recall seeing in this game, but again I didn't really find this to be an overly lewd game, especially if you are not looking for it. It can be played as just a normal none sexual game in my opinion. One the other hand, I mean paw, the music, which is made by Shizi, is also done decently well and fits with the vibe of each section giving the many different parts of the game its own personality in a since. These songs also make for some good ringtones as well, so I would suggest also getting the OST as well. The story of the game is simple yet I find the characters in the story to be memorable. But for the most part this is mostly a game play game and the story is mainly really simple and straight forward. As for the technical side of things, this game can run smoothly from start to finish without a powerful machine and I have never encountered any lag while playing this game at all on my laptop which isn't designed to run games at all. So I wouldn't worry about if this game can run on your computer or not, or the person you may gift this game to's computer or not. With that said, as I am writing this review, the minimal requirements are just that you have the window's operating system. This may update after I write this review, so you can check the system requirements for this game on steam yourself if you would like to see if it was updated at all.
Now this is all of what I touched on previously was just for the original Changed game and not the new Changed: Special Edition which has yet to come out as of me writing this review. But from the early version I played and from what I have seen online, the artwork gets better, there is new music added, there are way more animations and unique animations you can work for. The story gets a lot better as it is a lot more fleshed out. I would compare this by describing the original game's story as a skeleton or a first draft and the special edition to be more of a full story you would expect to read on a final draft. This game also becomes lewder as well, to the point where it become causally noticeable in some places. Still I would rate this "T" if you wanted me to give an age rating. The creator also has tweeted that they were making this version more weirder and I would say they have especially looking at the "Unity of Mind" game over for example. The game over name I took from one of the game's creators tweet about it.

I will end this by saying, if you like thinking about the motivations of the creatures you encounter & how this work than you may encounter a rabbit hole that hoes deeper than first expected.\t\t\t\t\t
