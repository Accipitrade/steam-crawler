This is the problematic part of the game. This game is tagged as "difficult", and that's no joke. The description mentions how this game is about 6 hours long. There are three main gameplay portions: puzzles, chases and boss fights.

Another problem is the random movement some enemies have. This sometimes means the solution for a puzzle will suddenly not work, or a chase might fail due to this. \r\n\t\t\t\t\tThis game feels like a run of Dungeons and Dragons where the Dungen master feels like playing against his players. Changed is a hard puzzle game where you die a lot and have to do the same thing over and over. Which is not a bad thing on its own. What killed it for me is that the game is against you in every non-fun way.

The biggest kill-joy is the unskippable dialoge and scenes. There are lot these. When you die, when you enter rooms, when seeing this for the first time. Most of these come AFTER a save point but BEFORE a puzzle / fight. You don't always get the option to run back to save after dialoge so you just have to solve the puzzle / fight while dieing many times having to redo the whole scenes and dialoge with no option to skip. It is okey for the first few times but it gets anoying fast. And will happen with many puzzles and "fights".

The other is the game mechanics itself. The game has build it's layout for enemies that move in repeating paterns. Hallways with a hole in the all you can stand in when something passes by. All these holes are dead ends. Most enemies do but for some reason some chase you while the layout isnt made for it. Wheter you make it or not is based on luck, where the enemy moved to when you were out of sight (aka when the monster is still out of you screen field). If it moves in your way you're done. The game does not allow to run around things, only to step aside of them.
The controls are not that great either. Mainly when sprinting, which you will need to do to get away from enemies. Sprinting in a straight line works, but taking turns not so much. Sometimes it takes the turn straight away, sometimes is take an extra step or 2 before turning. Lenght of sprint or underground does not seem to effect it, it seem to be random. You will die, fall in traps and all that a lot because of this. It does not make the game hard, it makes it annoying. Especially when you need to redo all the unskipable dialog and scenes again.

The game is not all bad. The music is nice. The style of the game is great. The story is interesting. You just don't get to appreciate it to much since the game is working too hard to annoy you. If save points came after dialog or if it could be skipped the game could be quite enjoyable. Hard but enjoyable. Right now, it is mostly just annoying.\t\t\t\t\t
