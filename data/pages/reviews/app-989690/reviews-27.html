\r\n\t\t\t\t\tThis is an early review, might add something later, but for now, I'm loving playing it.
This game format is a very well done mix between classic and new RTS games, and for me it suits just fine.
I know about this game fundations, but this game thematic appeal is very, very strong, and I can ignore this not having a complex story or campaign, because Im can find Conan's tales elsewhere if I want it. By the way, this rich universe already is told through all it's graphics detais in this game. The comic addition is very welcome although.
I have Pros and Cons, of course, things I wish could change. So let's go to them.

Positive points:
Pick up and play;
Thematic appealing;
Co-op is, probably, very fun (ain't played with, well, friends);
Very nice graphics and art direction;
Very extensive Tech Tree;
Challenging and rewarding, you always want to do better than the last time;
Very fun!

Not So Positive:
Bugs, like, units don't move when ordered;
Conan likes to ignore you when he is in a fight;
Tech Tree looks a awful, sorry, and it looks like it was made for a small and fixed resolution;
Tech Tree doesn't have descriptions and details besides the names.

Thing I Wish could Change:
Not so slow start (just a little more resources at the beggining?);
Tech Tree - yeah, it really bugs me;
Units reaction to the situation (they laugh at odd times)
Units easy access (like, on the top-left screen, with names and numbers, because it's hard to select one type in the middle of a really big fight);
At least one more active skill per character;
Another playable Conan - with different skill and visual (like the barbarian one).

Now, just want to say, I was playing my first game and on the first map, I thought that I have defeated all the five waves, and was waiting for a Victory screen of some sort. Almost never happened. Why? Well, the fifth wave never ended, then I started to explore the vast map, the, some time later I found a enemy soldier. He didn't move or anything, so I killed him. Then the final wave finally ended. Just telling the story of a bug that was nice for me, allowing me to explore the map without caring about waves. But it shouldn't be there, so, there's it.

I recommend the game to people that want a challenging and fun RTS, to play WITH friends, not against them (I love cooperative games). It's very, very fun to play solo too. And to Conan fans.

Note:
This is one of my few game review on Steam, in part because I want to see this game grows.\t\t\t\t\t

\r\n\t\t\t\t\tEdited Review: Now that I have played more I can give a proper review. While this is a very fun game to play, especially with a friend, it is not truly a game. This is the Survival Mode of a single player game. The "Campaign" is a 5-round tutorial leading you through the mechanics of the game, but that is all. Survival mode is generally an option on single player games because there is a certain monotony which comes forth after a few games. The lack of anything relating to a real campaign or single-player game makes Conan feel like the first release of a crowd-sourced game, a teaser into the mechanics and world of Conan while they finish the actual game. The co-op feature is wonderful, it is one of the saving graces of this game and a reason to continue playing after single player rounds have lost their flavour.

The devs have released several small patches to bugs still alive in the game, but there has been no word on any other content additions to the game. The game as released already has payable DLC. This leads me to believe that there will be no major content releases or additions for which the player will not have to pay.

For these reasons I do not believe I can endorse this game any more.


Original Review: This game is entertaining, but if it's worth the money depends on where the developers take this. It takes a good four to five matches before one can recognize and develop a strategy to enable your hero to explore. The game concept is very simple, surviving waves of enemies is as deep as it gets, however it still seems like there's nothing tying any part of it to another. If one can make it through 25 rounds one game, there is no carry over to any other game even in the campaign. Perhaps a cross-over system where your hero retains levels gained throughout the campaign would add a certain continuity to it and make it more engaging. However I don't know if there are any intentions to modify or add to this game aside from the DLC that released with the game.\t\t\t\t\t
