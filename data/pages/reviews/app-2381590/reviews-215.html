\r\n\t\t\t\t\tAll memes aside, the game is frustrating but fun. I will address the two most important things people complain about:

1. "The game doesn't have saving" - Now some people refer to a checkpoint system, which honestly I think would ruin the game, the whole scary tension part is knowing at any time you could fall right back to beginning. While others refer to the fact that the game should save your position when you exit, this point is valid (especially since the game can crash and you will lose all progress), however, the game is short enough to where I think this can be overlooked, I beat the game in 8 hours across three different sittings, with my last attempt lasting 2hrs 30mins.

2. "The mechanics are clunky" - Whether by design or not, I think the clunky mechanics add to the difficulty of the game, in a similar way to how the annoying hammer in getting over it feels clunky. I think the game plays fine, especially after the last update, they improved the vaulting and falling mechanics a bit. You will get used to how the character handles relatively quickly.

Now as far as actual criticisms go, my only one is related to the cheesy traps. There are a few points in the game where you will land on something, just to have it instantly fall and reset you back to the beginning (or some other similar trap). This to me isn't good game design and just feels cheap because there are no way to tell the traps apart or even react to them, you just fall, instantly.

Overall considering the price, I think it's worth a buy. Finishing the game is a huge satisfaction.

\r\n\t\t\t\t\tA fairly fun game, if you enjoyed Getting Over It or Alt F4, you should get this game. However, I can't recommend this game due to a lack of an auto save. Unless you're able to beat the game in a single sitting, exiting to the menu will have you start the game all over again. This was particularly frustrating when I closed the game about 4/5 of the way up to go to bed, only to find my progress erased the following morning. I understand the desire to avoid save scumming, but this can be avoided with a clever auto save system, while still permitting players to save their progress and pick up where they left off the next day.\t\t\t\t\t
