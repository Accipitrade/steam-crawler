\r\n\t\t\t\t\tThe game as a platformer is doing its job okay'ish. However, as many already said, the game lacks some proper QoL features such as checkpoints, especially since getting back up is so tedious thanks to slow climbing animations and poorly implemented music/voicelines.

While you climb the random stock asset tower parkour, you're constantly being bothered by a cheap AI voice that keeps bragging about how bad life in Favela is and how abusive the family of the narrator is etc. Nobody cares. It doesn't sound interesting and because it keeps repeating itself over and over again as you repeat the parkour after falling, it's just annoying.

This is being accompanied by some random triggers which play the same stock music whenever you walk past them, making you feel like you're watching some Bob Ross painting session.

Since everything is so barebones and barely functional, I'd say the guy who made the game is new to the dev scene and went for Unreal Engine for no proper reason because the game could have been great with any other, more accessible engine as well. The game is even using its template project name in the task manager so that says everything.

So as it is right now, the game is just not good. It looks great because of stock assets and unreal's lighting features but that's really it. The gameplay is average, it lacks some features and the poorly implemented voice lines and music are just tedious. Fix that and I'd actually enjoy the game.\t\t\t\t\t
