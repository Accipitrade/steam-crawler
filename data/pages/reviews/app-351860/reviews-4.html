\r\n\t\t\t\t\tCultures - Northland, actually "Journey to Northland", a game from 2002-2003, now on steam.

It started quite calm.
No setting options in Menu and only Fullscreen, well thats fine.
But don't you dare to minimize the game, or else the half screen of the game turns green/purple.
So we first spend 1-3 hours researching how to turn the game into borderless window mode, to find out what we actually could do:

Go: Library -> Cultures - Northland -> Manage -> Browse Local Files -> Saves -> in this folder you create "opt_glob.ini" and
write:
gfx_screen_width 1680
gfx_screen_height 1050
gfx_screen_depth 32
gfx_fullscreen 0

Now we can play.
First play the Tutorial, the Campaing doesn't give you a crash courses.
It took me a while to finish the tutorial, like 5 hours, it was so boring, there were a bunch of symbols i pretty much never had/got to use. And after Tutorial the excitment to play was gone.

You can get this game on sale for like 1$.

5/10\t\t\t\t\t
