\r\n\t\t\t\t\tA nice little collection of the better half of the Mega Man X series. X1, X2, and X4 are some of the greatest action platformers to grace the planet, and X3 is... okay. Has a nice gallery of trailers, music, and images. The new mode is a nice little distraction, but it can be a pain with some of the combinations of mavericks they expect you to fight so I suggest playing on easy even if you are a Mega Man veteran. You even have the options to play the original Japanese releases.

The game does away with all the save features from the previous collections for password saves. It will only save in-between levels now so this may be a bit of a turn off to some newer players. The game introduces a new rookie hunter mode as an option for newcommers to ease into the games, but I'm not sure if it's a better alternative than what the older collections offered for them. The game also has a slight delay with inputs; if you're familiar with the original games, you'll notice it. However, it can be accustomed to.

One last thing, the game has a tendency to run poorly on certain systems. I personally had no issues myself, but if the game has these issues for you, I'd recommend you refund it. Sad that I had to say the last part - these issues should not be a problem in 2020.\t\t\t\t\t
