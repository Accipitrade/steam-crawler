\r\n\t\t\t\t\tWould give this game a neutral rating if I could. I hate that one of my favorite series of all times (the megaman X series specifically) has a mixed rating overall on steam. I bought it anyway because I'm a diehard fan...but honestly it kinda deserves a mixed rating.


I've played through these games probably atleast a dozen times in my life and they are a blast. Some of the best platforming, run and gun action, and unique boss battles with tons of awesome character progression and powerups.

But...the implementation is bad. And I mean embarassingly bad. I've played on Snes emulators with much better performance and optionality than this. There isn't even a volume slider, so if you want to play the game with the volume turned down while you watch/do other things...too bad.

There are also some very noticeable frame drops/slow downs throughout, which is pretty inexcusable for a game that came out on the super nintendo and has been emulated flawlessly for far more than a decade.

Really just a piss poor implementation on capcom's part. But the game is still very playable and just as awesome as ever. I love this series. So don't take my criticisms at face value, if you have never played this series or you just don't have a better way to play, then this is worth it, but if you have other options it may not be worth it to you.\t\t\t\t\t

\r\n\t\t\t\t\tThere are lots of complaints about this collection but thankfully, I was one of the lucky people who were able to play this without any issues. The only problem I had was the black screen issue on start-up but I was able to fix it by editing config.ini. I had to change RefreshRate to 60.00Hz everytime before I played the game. It's still frustrating but at least that fixes it.

This collection contains the best X games out there. Whether you're a newcomer or a returning player, I guarantee that you'll enjoy them. X series went downhill after X5 so this collection cuts the action on climax which is good. It also has Day of Sigma OVA which was released with the PSP remake, Maverick Hunter X. Casual gamers might have problems with some bosses and sections since this collection doesn't have a save state feature so beware.

The only problem I have with the collection itself (apart from the technical ones) is challenges. Compared to Mega Man Legacy Collection, X Challenges are cheap and lazy. You just face two bosses at the same time and the story mode for it is a joke. There's also a secret boss fight and a "story cutscene" as a reward if you beat all challenges on hard but trust me, it's not worth it. Just look it up on YouTube. No need to frustrate yourself for such a worthless reward.

That's X Legacy Collection for you. Only buy it if you feel like taking a gamble because most people report FPS issues along with input lag. I didn't have them but there's no guarentee that you won't as well.\t\t\t\t\t

And yes, the game did slow down. I have a older laptop running on a quad core i7 with a 2GB Radeon 7650M.
It's not a powerful gaming setup but definitely enough to run such an 2D game.

Well the game speed depend on the fps, to achieve full game speed you need 60fps. But strangely on Megaman X4, for some reason on 720p it would stuck on 45fps (mean the game will run on slow motion 75% speed), restart the entire game may sometimes run on 60fps. Checking on After Burner it seem the GPU is under-utilize. Such symptom is unpredictable.

A good collection for Megaman fans but unpredictable gaming performance made it hard to recommend.

\r\n\t\t\t\t\tI've only played these editions for about 30 minutes, but I haven't found any trace of input lag so many others complain of. Other than the normal slowdown in specific spots that has always been in Megaman X. It was coded that way in order to fit the game into the SNES cartridge and thus remains in every version of the game I have ever played. These games play exactly the same as they would on my Super Nintendo.

And they rock. Some of my favorite games of all time.

Oh and the XChallenge mode is amazing! It pits you against 2 bosses at a time from all over X-X4. My first match up was Fourth Armor X vs Chill Penguin & Ice Walrus! It was delightfully intense. By themselves, these bosses can be bested with just the X buster, but together, they actually compliment each other and made for a tough boss fight.

Anyway, I rather enjoyed this iteration of the X legacy collection.\t\t\t\t\t
