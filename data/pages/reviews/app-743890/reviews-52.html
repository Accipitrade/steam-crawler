\r\n\t\t\t\t\tSo...

If you're looking that this collection you probably already know that the first four Mega Man X games are all fantastic. However I can say that this collection has a lot of performance problems. The first two Mega Man X games experience lag due to issues with the game's engine and this lag is present in ever version of the game including emultations however some people have experienced much worse problems in this collection than in the original and emulated versions of the games. However I can honestly say that I've experienced no such issues myself. I have not had any issues with imput lag except when playing on a steam link where there was about a half second of imput lag which made any moment requiring precision a ♥♥♥♥♥♥♥ nightmare. This may have been a issue with the steam link and not the legacy collection because I've noticed input lag on several other games as well.

Mega Man X2 had some lag when I was playing but no more than what I experienced when playing the original game on my SNES using an original cartridge. It's not a dealbreaker and only present during moments when there was a lot going on in the game. But, several people have complained about it being very laggy so keep this in mind. Another thing worth noting is that if you use an ultrawide or 4K monitor then you're going to have to put up with 16:9 resolutions and a maximum resolution of 1440P.

Based on my Personal experiences I can recommend the Legacy Collection, but other people have experienced a lot of issues with it so my personal experience may not reflect what you'll have playing these games. It may be better for the time being to get the Legacy Collection on PS4 or Xbox 1 if you want to revisit these games.\t\t\t\t\t

\r\n\t\t\t\t\tThis collection of the first four Mega mans is great. The menus are well presented, very intuitive and fast to load.
I played the intro stage of each game and it works just fine as expected. The slowdowns that are present sometimes are proper from the Snes and it does not bother at all. Mega man X4 rans very good like it was in the PS1. The only issue i had in this game was some screen tearing when jumping and in the cutscenes on the X4 but i fix it in the nvidia control panel - manage 3D settings - program settings and set the vertical sync to FAST.

In X1, X2 and X3 you can save data in the same way as the original game by keeping the code that shows on screen. It does not bother me , you just take a picture or write down for when you want to load again your progress. In X4 you can save up to 6 different slots.

I played with a wired Xbox One controller and i did not have issues with input lag. You can set up in the case of the first 3 games the RB and LB buttons and in X4 you can use the triggers. The performance of the games is great. The FPS counter was always on 60 fps, and as i mentioned before the slowdowns are part of the emulation and they are not actual frame drop issues. Also i had no sound issues.

The things that i liked from this collection are the menu options that you can access with the BACK button at any time where you can configure some main settings of the games such as monitor size, filter (i used type 1 because it gives a clean look of X), wallpaper (when you play in 4:3 so to fill the screen), Rookie Hunter Mode (increases the damage you make and lowers the damage you recieve) and to exit to the main title of the game or exit to the main menu of the collection. Also i liked the X challenge mode and the extra content that has such as trailers and galleries of photos.

One of the downsites i found was that you can only choose between 4 resolutions (1280x720, 1600x900, 1920x1080, 2560x1440). In my case i have full HD but this might be a big issuel if your monitor has a different resolution from the ones mentioned.

Overall i give this collection a 9/10 and if you are Mega man fan as i am this collection is a must pick.\t\t\t\t\t

\r\n\t\t\t\t\tIt's been nearly 25 years since I grew up playing Mega Man X on SNES US Version. First and foremost is that it MMXLC 1+2 comes with Mega Man X-8 which is super awesome!

Pros:
-Each collection comes with Mega Man X-X4 while the other collection comes with Mega Man X5-X8
-It has several different graphics filter to play which includes Smooth Filter, CRT (Interlace), and Standard, Vanilla Pixelated Filter
-It supports Windowed, Fullscreen, and Borderless Fullscreen Mode
-It contains art gallery, Day of Sigma Cinematic Video, Japanese Advertised Videos, Achievement System, etc.
-Rookie Hunter Mode is basically Easy Mode for novice players who want to experience Mega Man X Series for very first time which essentially cause your character to receive less damage, though not immune to spikes and thorns which results instant death kill.
-Game supports Keyboard, Xinput (Xbox 360, Xbox One, and Xbox Elite), Dual Shock Controller (PS4)
-Does not come with Save State and Rewind Feature

-Cons
-No keyboard or controller mapping options
-It has some slowdowns on SNES Version X-X3 which emulates similar to original SNES port
-It's separate collection, but I wish all X games are in one single game package
-I wish Capcom add Playstation and SNES Version for Mega Man X-X3 which changes audio mixer and it does include cutscenes and different soundtracks.
-You can switch between US and Japanese Version
-X-X3 is basically emulated ROMs which emulates the original version of SNES port which results noticeable slowdown, compared to Gamecube and PS2 port.

Other thoughts:
Although I have been a huge fan of Mega Man X games for very long, some PC users report that there are some bugs in the game. For instance, saving the game may sometimes lose progression for things like Sub Tank, you may have to start all over again from the beginning where you want to start at Sigma's Fortress even though he/she have completed second or third Sigma Fortress, etc. For performance-wise, some users have experience input latency and choppy framerate which I don't have an issue whatsoever. Perhaps, their specs may not have met system's requirements. Maybe Capcom needs to do better job porting MMXLC 1+2 for PC port by releasing new patch to address performance optimization issues. But all in all, I highly recommend this game for nostalgia MMX fans and new audience fans for the first time. Trust me, you're going to love it due to fast-paced action and dark, gritty atmosphere too. And of course, good plot and character development too!\t\t\t\t\t
