\r\n\t\t\t\t\tChildhood favorite game, first game(s) I ever rented from Blockbuster (#only90skidswillunderstand).

This collection is just alright. I feel like the GCN/Xbox/PS2 Mega Man X Collections did a much better job of introducing new features + integrating the emulation. The emulation here is VERY bulky, severe issues with frame rates when on my laptop (XPS 9560, i7-7700k w/ GTX 1050 4gb, 16gb ram, nvme ssd) however it seems fine on my desktop (Ryzen 7 2700x, GTX 1070, etc).

The way that controls are handled are weird and per-game, and they reset if you start up that game again...which is a tad annoying, but I guess its true to the SNES versions in that respect.

Overall, would recommend if you can't be bothered with emulating yourself, or you have a beefy computer and want to bask in some GLORIOUS nostalgia. Otherwise, Dolphin+emulation (or emulating the individual titles) may be more your stream.\t\t\t\t\t

\r\n\t\t\t\t\tI played through most of Mega Man X already and here's what I can say...

The Mega Man X Legacy Collection is essentially a direct emulation of the games from the SNES. Some may say that this is a bad thing because it doesn't innovate anything. I think this is a great thing. Not only is it keeping true to the original, it adds some extra options to change how it looks. If you compare this with a different SNES game ported to Steam, Chrono Trigger, Square Enix ported over the mobile version and forced the players to have the bad filter on it. That's why it had a problem with so many bad reviews. Now that Square Enix updated it, Chrono Trigger is slowly recovering from the reviews. With Capcom, they made the filters optional from the get go.

However there are a few gripes that stand out makes me think this game as an okay port to a bad port. Much like with Capcom's other title, Resident Evil 6, there have been report about people, including me, that say the game cannot run at times (Err09: Unsupported Function). In fact, when I first launched the game, it crashed immediately. I looked into it and eventually it resolved itself. Upon a successful boot, I looked into the options to change the screen resolution and it crashed with the same error code.

After tinkering around with the configurations file, I decided to reset the configurations and play the game in a windowed, 720 resolution.

Another small detail (at least in the first game) is that they kept the codes to enter, but they added a save button between each stage. I'm not 100% sure if I like that or not.

Overall: I'd give it a 6/10. There is so much to the game, but there can be more. Unfortunately there are bugs and I would like to enjoy it in fullscreen!

Controls & Handling: 8/10. The controls are great, easy to pick up. It is exactly how it was in the original. If it isn't broke, why fix it, right? This only applies to the actual controller handling. Keyboard, on the other hand..? Not so much.

Graphics: 8/10. The graphics are supposed to be two dimensional and blocky, but some times there are slow areas, especially in Mega Man X. There are a lot of different filter options for you to choose from.

Music: 7/10. They kept the original music, but as I stated earlier, it's an emulation. They didn't improve on anything.\t\t\t\t\t
