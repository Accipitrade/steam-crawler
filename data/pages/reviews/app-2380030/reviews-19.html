size={
\t\tx=1920
\t\ty=1080
or

size={
\t\tx=2560
\t\ty=1440
save and open game if it still doesn't work try and open settings.txt again
2. Find this gui scale: (This should be 1.0 or 1.5 you can experiment) This is just below the size thing we just checked.

gui_scale=1.000000
\tgui_safe_ratio=1.000000
or
gui_scale=1.500000
\tgui_safe_ratio=1.500000

If this still doesn't work it's time to experiment with the game and launcher options.
Based on what I experienced when I tried doing the options in game since I'm a two monitor user it's undeniably the most hair ripping thing I've gone through.
What worked for me though is changing the game settings into Full screen and choosing my display monitor into my main monitor which was (1) Generic PnP Monitor and my second one was (0) Generic PnP Monitor choose fullscreen resolution to my main so that when you start the game it looks not what my monitor resolution looks like but when I click out and then into my game it auto fixes to my resolution.
