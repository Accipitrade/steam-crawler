\r\n\t\t\t\t\tI've played over an hour of Pinball Wicked so far, and I can say I recommend it, with some caveats. Since getting an AtGames Legends Pinball machine, I've been on the hunt for any pinball game that supports cabinet mode. After doing some research, I found out this game does in fact support cabinet mode. You will need to request it from the devs, but it is very easy to do. I was granted a code within 48 hours of requesting it.

The game looks great! My rig is not top of the line, so I am not running ultra settings. I imagine those who are are in for a visual treat. Additionally, lighting effects like lightning, still look amazing on "High" settings. There is one single table. That may be a drawback for some, but I appreciate a well done pinball game. If you give me one solid table, especially for $9.99, I feel it is worth it. The game is still considered Early Access, but judging by the last posts from the dev, this has been abandoned.

Would I still recommend this? Yes, so long as you don't go in expecting any further developments. While this game may never leave early access, what is here is complete. Yes, there are some wonky physics here and there. But nothing egregious enough to take you out of the experience completely. If you can live with a single, well done pinball table, it is an easy recommend. Especially if you are planning to play in a virtual cabinet.\t\t\t\t\t

\r\n\t\t\t\t\tThe game starts by installing Unreal Engine 4, then asking the player to read the privacy policy and EULA which mostly regards the website for the game.
The user interface has a lot of options, buttons and slider bars, which doesn't feel great. Controller force feedback is active by default, but there is an option to deactivate. A few of the UI buttons can only be used with the DLC.
The game defaults for a 5-ball first game. Maybe it was just my camera angle, but the flipper action seemed weightless.
There are a bunch of features and animations, but they only confused me as to whether I'm supposed to shoot the lit targets or the animation. I suspect more familiarity with the game would answer this confusion for me.
At the end of a play, it asks for initials which are for the local score board, and then it automatically puts the player's Steam name on the global leaderboard.

Some games feel right and I want to spend time on them for a while. This one felt clunky, like if ARK had a pinball mini game. I'm not really interested in opening it again.\t\t\t\t\t
