\r\n\t\t\t\t\tThe Lords of Waterdeep is a fantastic board game. Played the physical game several times, as well as a Tabletop version.
This version is nice because it automates many aspects of the game that isn't possible on other digital versions.

But WOW is the user interface horrible. Possibly the worst gaming UI I've ever used. Each move requires multiple levels of confirmation, and the UI doesn't make it clear that it's waiting for you. That is particularly problematic because the game has an internal timer that will cause an automatic forfeit if you take more than 30 minutes to complete turns. Most of the time that gets used up is spent in trying to figure out how to use the interface. So yeah... you can find yourself forfeited *because* the interface is that bad. (top tip - change that 30 minute configuration to whatever the max is when you're playing with friends). Resolution support is terrible as well - I have an ultrawide monitor, and at the 3440x1440 resolution, you can see *less* of the interface on screen than if you run it windowed at 1920x1080.

If you really like the physical game and have friends that you can play with, it's worth it. If you're getting matched online and playing with randos, stay away.\t\t\t\t\t
