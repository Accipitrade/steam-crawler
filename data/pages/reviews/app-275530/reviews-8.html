\r\n\t\t\t\t\tVery funny. This is such a classic. I still have the retailer box for this game I played many years ago. When I saw it here on Steam I had to give it one more try just for the sake of refreshing my memories about it. It didn't disappoint. The game is an RTS with a frantic non-stop action. There are several units that you don't need to buy or gather any resources for or doing research and upgrades, you just need to conquer territories and you'll get units out of the facilities from time to time. Is kinda like a 'king of the hill' game.

Found no bugs. Runs fine on Win 7.

Games like this, classic good old ones, are truly entertaining games. There's no seriousness or pseudo-philosophical, political stuff, just plain crazy dumb fun. This game shines with the hilarious cutscenes that became almost legendary. Is not a very long game but good enough for the price. It also includes cards and achievements, so is also a good deal.\t\t\t\t\t

\r\n\t\t\t\t\tI got this game at an enormous discount, so I shouldn't be complaining. I bought it because I remembered an old advertisement from a long time ago which I saw in a gaming magazine. Always wanted to see what the game was about, and now I did.

I'd actually upvote the game if two things in it were possible - paning the map with the keyboard, and giving orders without deselecting units. Seriously, it's an old game even if this is a (rather nice) remaster, and if you give orders to a troop it automatically deselects it. Not being able to move around the map with the keys makes this extra infuriating. Those two issues make the game kind of unplayable, and if that was different it wouldn't need a 90% discount for folks to buy it.

There are pathing issues, too, as well as problems with giving orders to individual units once you bunch a few up. Fixing those would go a long way to improving the experience, too.

A for the game itself - it's kind of mislabeled. Folks turn it on expecting a great real-time strategy game from the era of Warcraft and Command & Conquer, and what they get compares to those badly. The thing, though, is that Z isn't really a real time strategy in the vein of those games but rather a proto tower defense. And if it didn't have interface issues it'd be a fun short one and quite recommendable at a discount price. Tower defense fans don't really have old games to trace their roots to.

As is, the game is unrecommendable and if you're thinking of snapping it up, check if by any chance the devs have read this and made adjustments. Otherwise, I'd only recommend it to achievement hunters with a few hours to kill, as nothing but the final level will take much effort.\t\t\t\t\t
