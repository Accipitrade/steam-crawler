This rts was way ahead of it's time. Unlike today's standard rts's of aquire minerals, get army and attack.
Z (or Zed) had you start with a handful of units each map, at which point you would have to secure flag points to a) get more resources and b) get higher level troops.

If you had to make a comparison, it would be similar to warcraft 3 (without hero's but needing to keep units alive) meets dragon commander (capture territories), meanwhile this is in a game from the 90's on ms-dos.

Though a bit dated, it still kinda stands up and definitely worth a try as it is usually not to expensive, if ever over $10 cdn these days.

i would like to note that the point and click interface is not as good as it was, hard to tell what you are telling units to do.
none of the keyboard shortcuts work. the only way to deselect is to give an order, that is a pain.
some automatic actions seem a bit off, for instance path mapping for vehicles is horrible. vehicles get stuck on the little habitat caves on the maps where they would originally just blow them up., which relates back to point and click, an attack/suppressing fire button would be nice.
getting units to occupy a vehicle when you have more than one unit selected seems to be a challenge also.
difficulty is very low, seemed to win lots of engagements that i shouldnt have, and tore through every level first time even after making some critcal errors.

the original had multiplayer, where is that?

a skirmish option would be good.
i love this game, i love the cut scenes and the banter, and usually the gameplay is challenging, and rewarding. if i had not been familiar with the older version and interface i think i would have found this version clunky and frustrating.\t\t\t\t\t
