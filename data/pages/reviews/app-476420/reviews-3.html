\r\n\t\t\t\t\tThe game itself is fine, but there are several issues I found with this particular implementation that I found far too irritating to recommend against other board game apps:

It is possible to permanently skip the tutorial, which is not good if you're a first time player, since the turns are highly procedural, and it's not always clear when you're allowed to do things or how. I had to stumble a bit before I figured out that buying cards and starting/supporting succession were mutually exclusive.

No Undo Button. A misclick can take you beyond the point where you're able to do what you wanted to do (either buy or support succession).

That made the experience of play unpleasant for me, just trying this out. Bad first impression.
As a nitpick, I was able to play long enough to draw the first campaign succession card, but when I tried to actually read what the card was by left-clicking, I kept getting put into an unskippable cutscene less than a second later. Right-clicking, which in the game interface is how you expand the cards, does nothing.

Overall, the game itself as I said is decent, but not spectacular, and the UI issues make it a net negative for me.\t\t\t\t\t

\r\n\t\t\t\t\tUpdating this review after 57 more hours played:

The boardgame is still fun.
The computer game is mostly the same and the game itself is also a lot of fun.
The interface still annoys me in quite some places though:

Noticeably:
1)
When you start the game it takes you to an unneeded (and unwanted) intro screen. In many games you can disable these kind of screen or introductions. There would be a config:/skip intro movie,
In HoC there is no such option and you return to this needeless screen after every game.

2)
After skipping an optically nice looking screen follows. On the right are a few options, all but on are ghosted (which generally means not selectable). And arrow follow the mouse and on unghosts. Again, pretty eyecandy. Harder to use then needed.

3)
While on this screen you are spammed with a few server messages. I also found no way to disable theese messages and these also always pop up when starting.

4)
After these we get to the good part. We can start a game! Yay.
This is not done by pressing "play", "start" or something opvious but a strange option called "versus".
I guess this makes sense int he language of the creator, but the english translations here is confusing.

Next up is'vs CPU', which is equally confusing. I would suggest "vs computer" rather then vs aa very specific single chip inside the whole computer.
Once again, I suspect untested translations, which can probably easily be changed in future versions.

5) Now follow the good thing. You get to play. And the ifself game is fun.


That would be a good place to end, but a few more notes and ideas:

a) Playing is done with a nice coliurful and random background. It looks nice, but there is no way to turn of this distracting eye candy.
b) The board feel cramped, even on 4K resolution. Sorting the cards in a 3x3 grid array would save space
c) Reading a card is done by right clicking it... for as long a you keep the right mouse button pressed. It would be nice to keep the text readable without continuous pressing the button.
d) The computer plays fast. They often end these turn and then you need to dig in what the 3 computer players now actually did. A pause after each computer player, and one whenever they effect you would be nice.\t\t\t\t\t
