Despite purchasing it at 50% off I'm still not quite sure if I received my money's worth. First, there are definitely some positive aspects of the game: The airports are realistic, it offers a challenge and despite me being female and not a native speaker the voice recognition works pretty well. However the base game only comes with 3 airports and all airlines have fantasy names - a mod to make the names realistic as well as more airports are only available as paid DLCs. It also is missing some really basic convenience features such as keyboard shortcuts to speed up, slow down and pause the game and there is no option to save your game so you have to start anew after every time you have to shut down the game. With this all in mind, I am still enjoying myself playing Tower 3D but it is definitely not a game I will recommend to a casual player and I only recommend it to airport enthusiasts if you can buy it on a 50% sale because it is not worth its money at its full price.\t\t\t\t\t

\r\n\t\t\t\t\tThe overall concept is great, but the product has some serious problems:


  • Way to expensive for a game which feels more like Early Access. Plenty of DLCs available (which is great) but it seems to me this is the only section where the developer resources flow. No changes in the base game whatsoever.
  • No savegames. No. Savegames. Oof.
  • Only basic commands are available. Would be much smoother if you can tell the airplanes to wait for a specific callsign before proceeding with taxi. Or tell some commands in advance like crossing a runway to save precious time.
  • Voice commands works 80% of the time for me, but when it doesn't...oh boy. Needs some configuration because it's using Windows voice command system which wouldn't be necessary at all? Feels a bit cheap tbh and surely doesn't help the recognition problems if you can't train the words that fail specifically.
  • So many bugs its really anoying

If you get multiplayer up and running it works. But you have to sit in the role you choose at the beginning forever, there is no switching option.

Overall the devs could spend a little more love on this game. The graphics are not super important, but don't expect anything fancy here. Its like 2002GFX all over the place.

Yes, there is potential, but at the moment I wouldn't recommend it.\t\t\t\t\t
