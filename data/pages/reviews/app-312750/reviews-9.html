The game installs, and starts fine, but this is where it gets tricky. The gamepad doesn't work on the start screen (where you see Play and Settings), nor does the trackpad. This is with the default controller setup for the Deck.

On this screen, you need to use the touch screen. Then the rest of the game you can use the gamepad. This is a small hitch, more of a user experience issue than a technical issue. However, it seems that the audio in the game stops working for certain scenes, then starts back up in other scenes. The game works, but on the Steam Deck it has its issues.

I would still recommend the game! Just be wary if you're playing it on Steam Deck.\t\t\t\t\t
