i would still RECOMMEND Trainz Plus for the new content features, but be ready to spend some time to figure out how to find the content you want.\t\t\t\t\t

Overview
Overall this is a really good game , it has definitely passed TS Classic with the new trains which have full cabin control, with even more like air conditioning, something TSW or TS Classic don't have.

However, the GUI is a bit outdated a hard to navigate in, and there's no camera shaking (it is currently the most requested feature so maybe soon?)

Conclusion
If you've been playing TS Classic and and want to bring it up a step, I definitely recommend this game, even if you've never played a train game this is definitely one of the better ones out there. Especially with the surveyor.\t\t\t\t\t
