\r\n\t\t\t\t\tLet me just start with the fact that I absolutely love this game and will still continue to play it. BUT here's the deal, if anyone reading this particular review just know that this game despite the overwhelmingly positive amount of reviews, is easily one of the most poorly optimized games I have ever played. Let me tell you my experience with this game and the series in general.

I've played this game series since FS13 and I have enjoyed every single game since then and haven't had any issues until FS19. My issues with this game is that I feel like I'm playing my game at bare minimum hardware specs, my PC has an Intel i7 4770K, an ASUS Strix 1070 (I have 2 of them in SLI but SLI is disabled for this game, more on that later) and 24gb of RAM, this setup should run the game absolutely perfect under any kind of conditions but this is where the issues start. I've heard that the game does not support multi-threading, what that means is that for the guys with multi core, multi thread CPU's is that the game is only going to run on 1 of those cores I believe, correct me if I'm wrong but in my experience my CPU sees at most 40% load despite low FPS, and same with my graphics set at med-high settings. Inside the game itself. Speaking of which, y'all players with PC's that have more than one graphics cards in SLI/Crossfire, this game seems to hate it and I saw a performance increase by disabling it, if I were you I'd keep that in mind if you're having FPS issues. Now if you have a large amount of equipment (for pretty much any large scale farming setup) the game is going to be dog slow because of poor optimization, the game is trying to load up every single part of an implement no matter how far you are from it and that really takes out the PC performance wise. And with large amounts of equipment in one spot, it can really eat up your FPS like its nothing. It seems that Giants is less interested in fixing the rather incredibly problematic issues with the game and more interested in bringing us DLC that a lot of us can agree that you can see better in mods anyway. So with that being said Giants in my honest opinion is wasting all their time on what they're currently doing. So until I see some performance enhancements in the game itself and I am able to see the 60+ FPS all of us Farm Sim players deserve, I cannot recommend buying this game in it's current state. It's too unstable, and with a large scale farm it can get rather unplayable unless you have a computer powerful enough to trip the breakers in your house.\t\t\t\t\t
