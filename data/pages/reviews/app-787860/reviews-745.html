\r\n\t\t\t\t\tI haven't played any of the prior Farming Simulator games, this is my first. It seems very nice, playable,. good graphics, semi-realistic mechanics, etc. I will say that the game doesn't make it "easy" for beginners. The help menu is good in offering the basics of everything, but lacks detail of how to actually do something. The tutorials could have the option to be more extensive, for example. The equipment could be a bit more understandable to match with other equipment, etc. I just get a feel that the game relied on players already understanding the games mechanics via previous versions and the new players are just left to fend for themselves. I'm not saying the game should be easier, I get it's a simulator, but some more in depth guides in game would help, so I don't have to keep switching the screens to get the info I need that really should be available in the game to start with.

If you are looking for a farm simulator, this is a great one. I believe my interest will be held long enough to learn the game, which is usually a big ask for me.\t\t\t\t\t
