\r\n\t\t\t\t\tMany, MANY hours on this game. I've also played FS17, and 15. All wonderful games, and I love how each title (although fairly expensive) is basically just an update to the same "platform." All of them use the same basic build, and everything (GUI, physics, etc.) is all relatively the similar. However, Farming Simulator 19 has definitely been the best. Amazing graphics, easy to use menus, and relatively simple, straight forward gameplay. Only a few things I can say other than amazing. Lacks a realistic transmission setup with gearing, which is really only an issue if you're extremely into detail. Other biggest issue for me personally (maybe others) is that, as I said, it's an extremely basic game without mods. Farming is it. Until you get mods. Mods are nearly required for this game, especially if you're into that kind of thing. Lastly, if you have too many things (machines, tractors, cars..) it lags BAD, which is to be expected, and can only be fixed through better PC specs (so fair warning). Other than that, wonderful game. Would totally recommend for anyone, so long as you have the ability to install mods.\t\t\t\t\t
