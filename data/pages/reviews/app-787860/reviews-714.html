\r\n\t\t\t\t\tI play this franchise since the fs13. You might think that Farming Simulator 19 is the best game from them all and.. of course it is. Has the biggest number of equipment and the best-looking.

On this game you can do agriculture, cattle raising and forestry. It is really a game that you really choose what to do: You like small fields with small/old equipment? You can do it. You like big fields with humongous equipment? You can also do it!
You start with only money, a farm and money or farm, equipment, fields and less money (difilculty modes) and try to become a successful farmer!

This game is very relaxing, it may look a bit boring but you always have things to do, specially when you have a big farm: Farm fields, take care of animals, selling trees, harvesting fields etc... And if you are a bit bored you can try the Multiplayer:

The Multiplayer is very easy to use, Steam FS19 can play with Epic Games FS19 for example but there isn't cross-play to consoles. On MP you can have all players on the same farm or have each one a different farm. There is Mods support (i will talk about this on the next point). Make sure you have uPnP enabled because if it is off some plays can't connect (it is the internet providers problem and not a game problem).

The community is very friendly. Now let's talk about the thing that makes this game alive: The Mods. Every week-day on the official GIANTS Software ModHub comes out new mods! Maps, small tractors, big tractors, harvesters EVERYTHING. It so easy to use mods. You can access the ModHub inside the game on the Mods menu. Install and already done. Or you can go to the website, download the zips and put them in Documents\\My Games\\ Farming Simulator 19\\ mods and done! I really think that mods are what really makes this game because it allows to have more things in the game (specially the maps).

The Price? Oh... I think this game is "cheap" considering today's games. It runs on almost every computer and very well optimized. 15 FPS doesn't feel like 15 FPS. The graphics aren't the best but it is enough. Try using shaders for a beautiful game! I own some DLC's. I don't recommend the Cotton nor the Rottne one (too much expensive for litle stuff or things that you will not do a lot). The one I recommend the most is the Alpine and the CLAAS one (platinum edition).

You might think that you would need a wheel or those farming stuff from logitech but I use a keyboard. Driving with keyboard is fine and since you have many binds a keyboard right in front of you helps. If you have a joystick don't hesitate to use it! It is very helpful when maneuvering bales, logs etc...

If you like nature, like a game to relax, learn new things or play with friends this is the right game for you.
