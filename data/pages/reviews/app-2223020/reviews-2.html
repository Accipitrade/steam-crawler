\r\n\t\t\t\t\tI never write reviews since I rarely feel like it but this game deserves so much attention! I really hope that more people will play it. I've only played one route so far but it's been amazing. I've laughed, I've cried, I've fallen in love with the characters. And there's still 5 more routes to go! Sure, the game seems to be a bit short for this price (one route took about 2,5 hours) but it's fully voiced and that alone justifies the price in my opinion. And on that note, the voice acting in this game is AMAZING. Same with the soundtrack. Seriously, Eternal Love Studios, you did a great job, especially since it seems to be your first game. I'm looking forward to more games from you!

I'm not sure how to get into the sixth route since there is only 5 choices at the beginning to choose a character to pursue... unless it's some kind of true end available after finishing the previous routes. But anyway, if anyone is hesistating to buy Glass Heart because of its price, don't! This game is worth every penny.\t\t\t\t\t
