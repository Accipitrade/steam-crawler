\r\n\t\t\t\t\tOverall it's a fun game, but it's got some terrible design choices that make it grueling to play at times. It doesn't feel nearly as ready as the version number seems to suggest, so I wouldn't pay full price for the game as it currently stands.

The single player campaign has several maps where you have to pull off a series of tricky shots. If you miss one, you have to restart the level. That's not very fun by itself, but the game is very clunky about restarting levels: the game loads the match over screen, then the title screen, then reloads the level. Watching the same animations over and over while trying to make a shot is really annoying.

There's also some really obvious bugs that still need to be fixed. For example, whenever you complete a challenge, you get a dialog with a template that reads something like "This is a challenge, description goes here".

I think it has a lot of potential: the dozens of weapons and scorched-earth vibe are really entertaining. Personally, I'd wait to buy until the interface is improved and more bugs are fixed.\t\t\t\t\t
