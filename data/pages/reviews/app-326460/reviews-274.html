\r\n\t\t\t\t\tA really fun game. I really love the graphics and the sound effects. It is still bit buggy though. I have a few suggestions for better gaming experience.


1. ADD THE ABILITY TO REJOIN GAMES WHICH I GET KICKED OUT OFF DUE TO NETWORK ISSUES!! Seriously this is a VERY BIG issue and I get frustrated each time it happens. Spent 20+ minutes in a 4.08x Rebound match got like 100xp and then Wifi/Ethernet has a problem for even a second. Result? I got kicked straightway! I've also lost a tournament which I was clearly set to win. This is getting out of hands. Please fix this.

2. The Portals pair should display some sort of visual connection. Maybe a fine dotted line connecting the pair. It gets incredibly confusing while playing Vortex mode as most of the Portals that spawns are of approximately the same size.

3. There should be an option to add Wind in Firing Range so that players can practice playing in wind.

4. While in Multiplayer mode, there should be better Sorting of the available Active games. Even better would be the ability to Filter modes based on the player's choice.

5. There should be better updation of currently Active games. Mostly, by the time I decide on a game to join, either the lobby gets full or the room gets deleted. And all I am left with is the annoying screen I mentioned in #3 above. Then I have to close the message ("Room is full") and enter the Multiplayer menu again.

6. If a weapon is shot at 0 sec. it disappears from the list and does not show up till the next turn comes. Most noticeable during Single tank fire mode.

6. Restarting campaign missions sometimes results in weapon already fired from the enemy/ally tanks.

Will add more points later.

Edit: (my original review points that got solved)
1. Ability to add friends from lobbies. I find some people to be more fun to play with than others. I would like them to add to a friend list of sort (after mutual agreement) so that when they are shown to be online I can play with them. This will lead to more pleasing experience. ------> They can be added as Friends in Steam by clicking their profile picture in-game while in lobby.
2. The transition screen ("grey split windows with ShellShock Live text written on three Gears and accompanied by 'Dhhmmm' sound" like when all players are ready and the game starts) should be changed/removed/muted. I find it very annoying and is an unnecessary effect, to be honest. I think many players would want it to be gone ------> It can be disabled in the Settings.\t\t\t\t\t
