The game is amazing, I loved how the game upgrade! I mean, I remember few years ago playing ShellShock Live 1 in Internet, and last year, ShellShock Live 2 in Internet too. The graphics of steam version are way better than the Internet version.
There are some options and bugs that have to be upgraded (however the game often receives updates)
My personal opinion abot the game:
In my opinion, this game is one of the best games out there, because is so complete. When I first entered this game I was shocked when I saw "Single-player mode". That's was a insteresting way to play the game (because in the Internet version you only could play Multi-player mode). I bought the game with a friend, so we play together, and we were amazed of how the games looked, because the weapons were so cool, and the graphics were AWESOME (specially Nuke, WOW)
But, the game has a problem, that if you play too much, you will get bored. So i'm waiting for new updates with new modes or new missions (for Single-player).

What i think this game should add, is Co-op Single-player mode. That would be so goood! I mean, playing missions like Single-player mode but with a friend, would be soo fun.
Also, i think this game could add some new tanks, but made by the community. Like workshop, it would be cool that people could make new weapons or tanks, etc.

So that's my review.
