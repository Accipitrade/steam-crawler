\r\n\t\t\t\t\tI've read some of the poor reviews on this game and I don't really get why people dislike this sequel so much over the original. Does the first installment have some charm that this one doesn't? Maybe? I'm not sure but honestly this is a continuation of what made the first one so good and it actually has some new features I quite enjoy. You can now build little platform bases on which you'll build the towers so you can get some added range which is pretty cool, it also allows you to build quick walls with the purpose of blocking paths. There are also areas of the map that are "locked" behind a resource cost which I think is a pretty cool way to add extra strategy. You often don't always need these annexed zones but it feels like an investment into your strategy when you decide to pay for them. All of the bonus mission scenarios are there from the first game which is part of what I loved so much about it, they designed the game in a way where you can replay the same level a half dozen times with adjusted parameters that require you to plan your strategy differently. It's honestly something that impresses me with the game design, the developers were able to design a game that you can easily sink dozens of hours into by simply tweaking existing scenarios. It's that fun. I think fans of DG1 will be happy with DG2.\t\t\t\t\t
