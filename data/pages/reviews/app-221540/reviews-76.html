\r\n\t\t\t\t\tYet another edit: Defense Grid (the Original) has more than convinced me this next game should be good. I am surprised to say I think this game will be worth buying at any point in time, especially if it helps the developers out.

Quick edit: if you have never played the first game, this might be dissapointing (like in my original review). I have since played the first game, and think this game has serious potential. I still say stick around as it develops, but wait to buy it. Also it doesn't crash if in windowed mode, so that's good enough for me.

Original: I have always loved Tower Defense games. I haven't played, or heard of, the first version of this... but I bought this anyway.

I know it's in early stages still (which some reviewers don't seem to get), but it's costly for the content that is just not there yet. Does this matter? No. Mainly because of the good reputation this developer has and the great fan base supporting it. It will pay for itself as updates come :)

Crashes are frequent in fullscreen. Maybe my computer, but other mid-high end games run no problem. When it works it seems rather boring in comparison to other TD games... again, maybe there is a lot more to come. I just think the graphics are not very catchy, the music is rather annoying, and there is no engagement in any story so far. I don't need story for TD games, but so far the game is just a few towers shooting the same few creatures.

I still have hope for this game and can't wait to see where it goes. I do like the setup on how buildings are bought and upgraded. Menus are designed very well, especially if you need to buy/upgrade towers very fast on a dying whim! But the crashing and current lack of excitement make this game a no go for me. I hope to see that change.\t\t\t\t\t
