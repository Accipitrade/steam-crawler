\r\n\t\t\t\t\tNote: I have only played single player campaign so far, no opinion on the MP aspect.

In a phrase, the game is "great mindless fun".

Technical Issues aside, I am not sure why there is a lot of hate on the forums for the game. At least the user reviews "Very Positive" atm seem about right with my feelings on the game. A quick note on the technical issues, my system isn't that beefy to today's standards, my rig is about 3 years old, and still using a GTX 680. On the initial launch, the spash screens themselves were stuttering. However, the note on the store front page asked users to update their drivers, and this did the trick. Game runs silky smooth now.

Pros:
- Fast movement. (Open beta you ran kind of slowly, but this is not the case with campaign at release)
- Great visuals, even with a GTX 680, no clipping, screen tearing, visual pop-in load times. Silky smooth so far
- Fun, fast gameplay
- So far the maps I've seen are nice and complex, and I can see how there can be certain chokepoints, sniper nests, etc if the same maps are included in the MP game
- Loadouts - this is a pro for me. Not an in-depth load out system like what you would find in Battlefield or CoD, or some silly rank-up perk system, but enough custimization with weapons and armor to make it relevent and interesting
- Responsive gunplay. Seriously digging it

Cons
- Music when enemies are around, sounds like death metal. Too loud, too obnoxious and doesn't suit the atmosphere of the game, it does break the immersion
- Menu UI (game play UI is good) is a little unintuitive and not completely suited for KB+M; i.e. have to use the mouse to click on a scroll bar for additional text, instead of being able to scroll wheel or space bar down
- Large download, The size is 44 gigs, and if you are on a slow connection, it will take a decent chunk of time

Is the game worth $60? Hard to say, and I'm on the fence about that. To me it boils down to how long the single player campaign turns out to be and how good the multiplayer is. At the moment, I feel like it should be in the $30-$40 range, however, that may very well change as I progress through it more.\t\t\t\t\t

I was expecting this Doom to be like the other Doom but it exceed all my expectations. I loved every second of it. All the demons and the iconic guns have returned. Plus with collectibles and stuff which is cool.

Things You Should Know:-
+ Good replay value
+ Finishing moves ( Glory Kills ) are so smooth, so good, and when I do it, I'm grinning gleefully. When I did it to a Hell Knight, I almost broke my mouse because I felt like I'm actually doing it.
+ The music always keep you on edge with heart-pumping-sicko-metal-beat EVAR !!!
+ Fluid Combat ( Silky smooth and no lag, no stuttering even in the heaviest and thickest of combat )
+ A monster GPU isn't required to run the game ( midrange class ) will suffice but of course you won't get to select the best/highest visuals )
+ Game is very well optimized and it feels like a PC exclusive game ( even though it's not )
+ Cool Easter eggs and collectibles ( Secret DOS Doom levels and figurines )
+ Awesome graphics ( Something like wolfenstein-ish a bit since it's the same developer )
+ LOTS AND LOTS OF DEMONS TO KILL ( I recommend playing it on Ultra-Violence )
+ BIG BIG GUNS ( The GUNS are a lot of improvement and I like the way it handles )
+ Guns Have Mods Yo ( Which I think is awesome and you can change them on the fly too )
+ I haven't finish the campaign but I can say it will be very long ( Something Like Doom 3 but depends on the difficulty too)
+ Survival Horror has always been my favourite ( Like L4D, Dying Light, Dead Island style )
+ SnapMap is a new thing they implement and it's really easy to use ( even if you are a noob )
+ Multiplayer is fun as hell and easy to understand and it's pretty fair too ( This is the main factor that I loved to death )

= No reload button is a bit confusing for as I always press R to reload which always change my weapons now instead
= A bit bug here and there but nothing that will distract me from my killing >:D

- Hefty Price ( $69.99 (SGD) is pretty expensive plus another $49.99 (SGD) for season pass is a bit too much )
- Those who haven't bought the game, I suggest you wait for sales if possible.
- Can't think of any other cons other than some bugs

Conclusion:-
All in all an almost "perfect" game. I would definitely recommend this to my friends.
Last but not least, a small shoutout to all DOOM fans,

\r\n\t\t\t\t\tOK so a quick review on how the game is doing for me at this point! (around 7 hours in)

First things first, the game is amazingly optimised, I dont have the most powerful PC in the world but I can run it all at ULTRA with no dips in framerate what so ever and that is really good to see in a world where most triple A titles comes out half complete or can barely run on a PC at all.

I have given the story a bit more of a go now and can happily say that it is everything DOOM should be, its fast paced and bloody with no room to breathe at all which makes it ultra tense and also so much fun to play, have only played 2 levels so far but from what I can see this game is shaping up nicely, hoping for more variety in the areas so it isnt all just Desert and Factory but what is there is polished and adds to the vibe they are going for.

I have however given the SnapMap a go and it is superb, created a map in around an hour and didnt even need to use the tutorials which are provided, this has been done in such a way that it is easy to pick up and start making maps of your own that can be played by others, the set up for playing other peoples creations is made really simple and you can go through a long list of created content and find what you feels suits you

I havent dived into multiplayer as of yet but will be doing that a bit later tonight once my team of mates is on so I will review that when I have given it a fair go

So far it is looking good and I would give it a strong 9/10 from the first impressions I have had, hoping that it can stay to this level of polish and fun for its full run time.
