\r\n\t\t\t\t\tDoom is INSANELY fun. From the beginning of the game just using a shotgun and a pistol on demons, to the end with your endless arsenal. Running and jumping around arenas, dealing with tons of different kinds of demons, and sometimes even just exploring has been some of the most fun I've ever had in a shooter. There's no good way to describe the pure fun that is had when playing the game, especially if you get it on sale for as little as 10 dollars. My only complaint with the game would have to be that all of this only applies to the single player. The multiplayer just feels like any generic shooter's multiplayer (if you ever even find a game that is).

Now we get less into complaints about the game itself, and more about this specific release of it. There are constant technical problems, and so many that I have trouble keeping track. Sometimes you'll get storage errors that tell you to clear 0 space. Sometimes a save file will get corrupted, FOREVER. Sometimes Alt tabbing crashes the game. Sometimes doing anything will crash the game. The game can't be recorded by OBS and most other screen recorders, so trying to stream or record this game is a nightmare. Sometimes checkpoint reloads will send you back to the main menu. This PC version is so bugged and ♥♥♥♥♥♥ sometimes that I'll just be too angry to continue playing the game. Some of this game's problems have well documented fixes, while others that I've listed here seem to be completely unheard of.\t\t\t\t\t
