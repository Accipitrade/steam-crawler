\r\n\t\t\t\t\tI really, really enjoyed this game at first. It has a lot going for it. One of the first games that I ever played was Doom. It was a lot of fun back then and the current incarnation is a lot of fun too. However, there were a couple things that killed this game for me.

One, I am a home row typist. My left hand reaches for the [F] key and I have <b>NEVER</b> used [W]to move forward in the history of my life. It has always frustrated me that we went with WSAD vs. EDSF, but I can live with it. I only need to make sure I config the key bindings to my liking. While Doom does allow this, it does not let you do this in multiplayer. [E] is locked to Player List. This kills multi-player for me which wasnt an issue in the beta.

Two, some jerk at ID software thought it would be cute to destroy someones save file if they used God Mode. This is what you get when a developer decides to use Checkpoint saves vs. manual saves. I was tinkering with some settings when I bumped the [~] key and opened the console. This brought back memories and I thought, I wonder if God Mode works. A few minutes later and BAM. God Mode. But, there was nothing fun, or nostalgic, about this. I didnt get to fool around in God Mode for a bit and then get back to my game. No, my save file was wrecked. I disabled God Mode but it made no difference. For whatever reason my save file became corrupted. Screw you ID. Its a <b>GAME</b> and you dont need to police it.

Because of those two things this game is ruined for me. I tried getting into it again, but I hate this game now and it is not fun.\t\t\t\t\t
