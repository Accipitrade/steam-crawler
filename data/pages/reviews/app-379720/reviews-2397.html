\r\n\t\t\t\t\tSo I am usually a multiplayer only guy but THIS taught me that I miss singleplayer games waaaayy too much. Seriously, this is good. The guns feel satisfying, cool characters, the story, while not all that much of a story, is kinda there and not convoluted or overly dramatic at all and I think this describes the whole game.
Here's your gun, that's a demon, kill it

When the game told me that there are "Extras" to find I was like "Yeah, probably really obvious ones and really easy to find on the way" but HELL you actually have to search for them and there are many, some really cool like the classic Doom levels, some meh if it's just some armor or something.
I only have two problems (the first one's pretty minor though) with the game and those are:
-The german version has some weird translations or dialog doesn't fit with the animation. There are also some grammar mistakes from time to time but eh, not gamebreaking
-There are some bugs here and there and you can very well get stuck if you try to go somewhere you're not supposed to go and you'll have to wait for a demon to kill you or you just kill yourself.

Other than that I actually just have wishes for the next one which are:
-Some more enemies to fight. I mean there were plenty and it's way better than "generic military soldier #2314" but about two thirds through the game you know all the enemies, which at the same time is a good thing since I like having o switch weapons for more efficiency like switching to the super shotgun against the knights
- Some more freedom would be nice since even if you can "explore" and find the extras, it still is a really linear game for the most part which again is a good thing because it's DOOM and things don't need to be complicated but even if it is only for scenery watching I would love to be able to jump around a bit more. Now I don't really know if this would be a good idea because it is DOOM and the fighting and challenge (if you want it of course) is the greatness about it so eh, if we get it I'll be happy, if not I won't make a contract with Satan to destroy ID Software and Bethesda

All in all fantastic game\t\t\t\t\t
