1. I am very Average when if comes to FPS games.
2. I have only finished about 2/3rd of the campaign by the time of writing the review. (edit: I

3. I played it on Ultra-Violence (Hard Mode) difficulty.
4. The PC I played it on had AMD R5 3600, RX 5600XT, 16 GB RAM.

>It is for sure worth the price, it is cheaper than a flagship game.
>Tho the game isn't priced flagship, it sure feels like it.
>Get ready for a challenge unless you play games on Easy mode.

>Not grind-y.

Some bad things:
>It does have a few levels where you need to "master" some mechanics.
>IF you use the Vulcan API for AMD graphics card, YOU WILL experience a few problems:
1.The game defaults to NVIDIA's API, so you will need to change that or you will get like 5
FPS in the menus and cut scenes.
2.IF you switch the game window while in a loading screen, ANY loading screen, the game
window will close.
