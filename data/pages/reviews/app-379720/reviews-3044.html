\r\n\t\t\t\t\tI bought this game during a 50% sale for 30 bucks.

Pros:

+Great and varied level design.
+Gripping fast paced arena battles.
+Stunning optimization thanks to Vulkan/OpenGL. Aside from the outside parts of game I always had 100+fps @ 1080p (Ultra) on GTX970/FX-8370. Great to see a dev work outside the DirectX monopoly and make it work this smoothly.
+Slick graphics (no pixelshader overload like in many dx11/12 games).
+Weapon- and player leveling system.
+Very precise and responsive controls.
+Challenging campaign. I played the game on Ultra-Violence and unlike so many other singleplayer shooters you can actually die.

Cons:

-Mandatory online DRM. While I had no trouble running the game it's still a big negative for me especially because they decided to patch it in long after the release without any heads up.
-No "New game plus" and you have to create a new savegame and start from scratch if you want to move up in difficulty.
-Different clients for single- and multiplayer for which you have to set up everything twice. Also you have to start the singleplayer first before rebooting the game into Multiplayer... Welcome to 1998
-Multiplayer is dead in this game and it's dead for a reason. It was not created by id software but a 3rd party hired by id and feels like a completely different (worse) game. There are 2 modes and both are a disgrace to the campaign and the franchise in general. VS feels like a Quake dev team stopped working half way and "Snapmap" is a failed attempt of creating a hub for player made maps in a rudimentary ingame editor.

Conclusion:

While I really enjoyed the campaign and finished it in 2 or 3 sessions it's hard to put into words just how bad the multiplayer is. So if you're looking for something to play online this isn't it. Also this means don't buy the season pass since all of the DLCs are multiplayer related. The game is worth 30 bucks not 50. Buy it when it's on sale, have tons of fun finishing the campaign, ignore the multiplayer and never play it again.\t\t\t\t\t
