This game was great while it worked. I've played the original DOOM while it was new, and this was starting to give me the same feeling of a really great game.

However, it didn't keep on working. In late May, it started crashing back to the desktop with no error message. Soon I discovered that this is a known problem after you've played for a while. Bethesda has set up a forum for posting crash reports, so they know that there's a problem, they just don't know what it is. That was in early June - they've had countless reports by now, but there has been no patch.

The game is unplayable for me, so I can't recommend buying it. If the bug is fixed, I'll change the review, since it really would be a great game if it wasn't for that one problem.\t\t\t\t\t
