\r\n\t\t\t\t\tAfter beating the game 15.9 hours with a corrupted file failure, it's time that ill tell you about DOOM.

POSTIVES
- Rip and Tear demons to death.
- Badass weapons.
- Fast-paced and energatic gameplay.
- Ultra Graphics (and a good game).
- The most interesting story I had in a SP campaign.
- Snapmap is a great mode
- I usually have fun in Multiplayer, otherwise, i'll let it pass.
- Recent update 6.66 showed how much did the devs care for the game.

NEGATIVES
- Repetitive areas which are really not worth it.
- My save file got corrupted when i was in the final level.

Verdict: 9.2/10 Amazingly Good

Curator note:
"DOOM is the kind of game which is all about the action and less about the cinematic. If you like killing things, this game is right up your alley."\t\t\t\t\t
