\r\n\t\t\t\t\tI really wanted to like this game but even buying it on sale it wasn't worth it. The game very much feels like a rushed VR demo released in the early days of VR game development.

The plot is really weak, which for a VR game, is almost to be expected. The weak plot would be completely excusable if the game was fun but, while it has a few fun moments, overall feels like a chore. The mechanics of the game work so poorly and it just seemed like they just dabbled in everything you could do in VR without actually making any particular part work properly. For some reason in random aspects of the game they decided it needed as realistic as possible. This is evident is things like using any two handed weapon or climbing. I honestly wouldn't have written a review if it weren't for the amount of climbing they make you do; it is extremely clunky and they use climbing as a filler to make the game last longer.

if you do end up playing it on a vive, browse the workshop for toggled grip controls so you don't end up with sore hands from having hold your grip on everything.\t\t\t\t\t
