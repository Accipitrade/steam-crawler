\r\n\t\t\t\t\tSeems like a cool game, motion is pretty fluid, gun play seems alright so far, but I ran into an issue with the shop part where you buy weapons.

I currently have only a baseball bat and 2 bricks, this is an issue because you need a weapon of each ammo type already to change to the ammo type you want to use as your currency to purchase weapons. Picking up ammo boxes will also change your ammo type to the ammo you picked up, this likely cause the issue where I have rifle rounds selected but not enough of them to purchase any weapons, it doesn't even switch ammo type when you run out (which makes sense in the context of reloading your weapon, but doesn't when using it as a currency).

This seems to be a pretty big oversight from the devs, it's really easy to run into because from what I've experienced so far, you lose all your weapons when you load into a new level, so every level you're going to have to try to find a box of the ammo type you wish to use as your currency, before you can actually use it as currency, or remember to not pick up ammo if you want to use your pistol ammo in fear that you might be picking up rifle rounds.

This seems to have been an issue since the game's launch in 2019 according to existing threads about this problem, but has been continually ignored and left unfixed. All they need to do is add a button to switch the ammo type... I really don't see why they still haven't been able to do that after nearly 2 years.

The game is good though, don't get me wrong, I really love the physics, the movement is really cool and is probably the best I've experienced in VR. They get so many things right here, but this one small issue is so frustratingly easy to resolve, I just don't understand why they haven't. It's not like they've forgotten about the game either, you can see in the update history that they're still making content for this game.

I long for a way to change my ammo type. Please, please add a button to change my ammo type.\t\t\t\t\t
