Weeks before this game was released I did what I could in other games to build up a resistance to the VR sickness I get from non teleporting movement. I didn’t get 100% but I thought I was ready for this, and somewhat I am... except I can’t go long enough to last the marathon between checkpoints!

Eventually I did make it though the first main level (first areas with the null men), and once you know what to do it’s not that long, but until you figure it out + dealing with being sick it can be nearly impossible to complete.

The weird thing, the game itself reminds you to take a break when feeling sick, yet offers no way to return to where you left off. What was the thought process here to deprioritize a save system? Like the devs are aware enough that people get sick and need to stop, yet fail to implement something to accommodate it? I _could_ leave the game running for the hours it takes to recover enough to get back to it, but that is not an acceptable work around.\t\t\t\t\t

\r\n\t\t\t\t\tHaven't played much due to busy home life especially with the holidays. I can say this though...

Playing on Oculus Quest via Oculus Link. My CPU is a Intel i5-8300 @ 2.3GHz which over clocks itself up to 4GHz which in my opinion handles the game great. On the main menu its in bright big red letters saying my CPU is not recommended. It handles it just fine, smooth as glass. GPU is a NVIDIA GeForce GTX 1060 with Max-Q Design and it handles it just fine also. I searched for a while to see if this setup would play Boneworks but couldn't find any close answers. So if someone else is in the same boat as I once was, I hope this helps.

Overall, so far, great game and great new use of physics.

Edit1: so I have gotten more time to play and my review stands. However, this game is in serious need of quick saves, check points, some kind of mid level save. If you're prone to motion sickness, making it through a level may be tough. Roughly an hour to get through a level. If you decide to bail early because you're not feeling good, you'll have to do it all again.\t\t\t\t\t
