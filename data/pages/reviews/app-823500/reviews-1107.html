\r\n\t\t\t\t\tNot sure if I should recommend this game or not... Personally, I don't like it.

The game itself is good, but it could be so much better, so much more enjoyable to play.

Here are the 3 main issues :
- The thing I hate the most about this game, is the added intertia on heavy objects : If you pick up a heavy hammer, you'll have a lot of troubles to control it due to the added inertia. This added intertia also makes climbing so annoying, which is a big part of this game, it's in my opinion a worse implementation than Robinson the journey. Some people like it, I don't, I think it would be perfect if we could disable "inertia" in the settings. The devs tried to add realism, but common, it's a video game, you're not lifting heavy objects in your VR room, you just have two lightweight controllers, it should be snappy in game.
- Big issue, but still annoying : It's badly optimised, in some zones I get a very low framerate, sometimes 10fps, sometimes 100fps, and my PC isn't bad actually, GTX1080 and i7-6800k. There's even a setting to automatically lower the resolution of the game when the framerate gets too low, a settings I have never seen in any other games, I have tried it, and it helps keeping a good framerate, but imagine playing a game where the resolution switches from 2k to 1080p/720p because you reach a zone where there's too much going on... so I disabled this setting again.
- Minor issue : Not enough checkpoints, it would be great if we could quicksave, some levels are really big and I don't want to restart these levels from the beginning. From what I've read, that issue will be solved later

At the current state, it's not my favorite VR game.\t\t\t\t\t

\r\n\t\t\t\t\tI want to love this game so badly. It is fun, the story is good and meta enough to be interesting. But for the love of god why would you release a game that you can't save? You unlock the levels but there's no checkpoints within the levels, no manual save points, and no item carry-over from mission to mission. The realistic physics are cool and all but climbing and pulling yourself up is so clunky, there really needs to be some assistance for these things. My game has crashed twice when playing with the physics and it makes me never want to open the game again because I know I'll have to start that mission all over. And the gun play is fun, but reloading is so frustrating when there's swarms of enemies. Why not just copy Pavlov's style? This game has ton of promise and I don't doubt that these issues will be fixed soon. I can understand outlier crashes and physics being fine tuned from player feedback, but releasing a game without saves is just mind blowing. If this is a traditional 2D game this would be a disgrace but people are overlooking it because it's VR. I don't understand why saves would be handled differently from 2D to VR so as far as I'm concerned it laziness.\t\t\t\t\t

P.S. Let me add something - this game has amateur level design, sound design and narrative design. This should be early access at best. Core gameplay loop is great, just like physics and gunplay yet it cannot be called a finished product. It has a lot of potential, especially if modding tools are released to the community but as it stands right now - I cannot recommend it.\t\t\t\t\t
