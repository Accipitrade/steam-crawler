tl;dr, good game but outdated, play it in VRChat, buy it if you wanna support sep-neko-ya.

EDIT: ok well RESONARK is still a fun VR rhythm game (but not really desktop-friendly) so i changed from a negative to a positive review, but don't buy RESONARK X unless you only want to support sep-neko-ya because RESONARK X is outdated

RESONARK X is a pretty cool VR rhythm game where you don't entirely have to rely on precise timing for rhythm. I am unsure how easily it can be recommended to people due to the 6-lane vertical-scrolling rhythm game (VSRG) gameplay, similar to other rhythm games like Guitar Hero and DanceDanceRevolution, but obviously entirely different due to the hexagonal layout of the lanes.

I don't really know how to describe it any more than that, and I do not feel like I need to; unless you have very good reasons to not be playing VRChat, you should just be playing the three RESONARK worlds in VRChat. They are kept way more up-to-date than RESONARK X, even adding in 2-player support and letting keyboard players read the hexagonal layout (albeit with clunky (IMO) default and unbindable controls), whereas in RESONARK X, desktop mode gets a very generic 6-lane layout. (They even got new desktop camera layouts that look nice!)

The last time I played RESONARK X (almost 2 weeks ago from this review), it felt... mundane. The menus' UI is not very intuitive or as pretty as the ones in VRChat. you can only use VRM models (as far as I know), and it's completely single-player. On the brighter side, all 3 RESONARK versions are contained in one RESONARK X package for convenience (as opposed to jumping between the three VRChat worlds), but the lack of any social features (besides some Twitter stuff?) along with the clunky UI makes RESONARK X not worth it.

Despite all of this, I do not regret buying this game. sep-neko-ya has made an interesting VR rhythm game, and if they could somehow incorporate ideas from the VRChat worlds into this game, this game could almost be up there with Beat Saber. As it currently stands, however, RESONARK X is not worth the time, and you should check out the VRChat worlds instead.\t\t\t\t\t
