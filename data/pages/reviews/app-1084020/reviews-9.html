\r\n\t\t\t\t\tIf you want to build something fancy here - you will have to use mods, that are called plugins here. Also, this game draws all the graphics on a single CPU thread. So the entire CPU is loaded at 30%, GPU is not used at all - and you have like 15 FPS when zoom the map out. Magnificient resource usage.
Some decorative sprites do not match anything. For example, there is a decoration of a light pole on grass, however, the grass for some reason has a different color from default grass. It's darker. There is also a 'beach' decoration which is meant to be placed on sand, but whatever the reason the beach's sand is notably lighter than the default one. Why the hell the colors are different?
There are also buildings that have roads. These roads are decorative and cars can't drive on them, but you can't even connect real roads to them, because these roads are placed BETWEEN two tiles! Who thought this is a good idea?
When you scroll the ingame screenshots gallery, the game does not unload the screenshots that you've scrolled, so it will run out of memory and yield a memory error. Epic.
Music is nice though.\t\t\t\t\t
