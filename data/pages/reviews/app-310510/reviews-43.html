\r\n\t\t\t\t\tThis game is annoying. Annoying because I love RPG's, and I love Tower Defense games, but it's just too unpolished still which makes playing it (especially with a gamepad) a frustrating experience. Some of the issues that need to be addressed:
  • The controller experience is absolutely terrible, especially when navigating various UI screens. The icon that tells you what to press is too small, doesn't stand out enough, and is never in a consistent location which means you end up hunting to find the button that you're meant to press.
  • The UI in general is pretty bad, with dialogs and detail popups frequently covering other controls or even in some instances important things like tower range rings.
  • There's no way to exit back to the map, so you have to force quit the game if you accidentally select the wrong level.
Seriously, it's so bad that it should basically be redesigned from the ground up... if any UI design conventions were followed, they were all the wrong ones. Neocore, do yourselves a favour and hire a User Experience Designer (or if you've already got one, a new one that actually knows what they're doing / doesn't drink heavily on the job).\t\t\t\t\t

\r\n\t\t\t\t\tIf you like Tower Defense or RPGs, this game is a must have!
The only game that is remotely comparable to Deathtrap is Sanctum2, where you can also join the battle after the building phase.
But Deathtrap has much more depth and customizations, you get loot, you can buy and craft items, you get xp and pick skills, just like a "real" rpg. And so far I have only played one of the three character classes...
Each Trap (=towers) has 5 different attributes that can be leveld up, so you have to specialize on something. Theres probably a lot of replayability, especially with the co-op multiplayer part, I havent even tried yet.
The only initial disappointment for me was, that you cannot freely place any trap anywhere. There are fixed positions on which you can place a certain class of trap, and you cannot block paths of the monsters. However, it turned out to be a very good design choice, it makes the game more fluid than this endless boring maze building that you have in almost every tower defense game.
I rarely review a game so early, but even though I've seen only a fraction of the content yet, it was already worth every cent.

EDIT after >50 h gametime: Still hooked by this game. Probably the best co-op mode of any tower defense out there! A few things still need to be balanced, but overall the game is huuge. Now, exucse me, I have to get more gold, loot, xp....\t\t\t\t\t

\r\n\t\t\t\t\tAfter playing The Incredible Adventures of Van Helsing I & II coop with my <3, Deathtrap was actually a brilliant surprise. The RPG games also by NeocoreGame had a simillar element just like Deathtrap when defending Van Helsing's Lair. Every time we got another quest that would sent us back we hoped it was one of these, so of course the moment I saw Deathtrap on the steam store I had to get it for both of us and since then, even so it was a pre-release game until yesterday, it was one of the best pre-release experiences I had so far. Content was added frequently, I've not seen one obvious bug in the whole time and the map creator is awesome.

I'd recommend it to everyone who already loves the Van Helsing RPGs and enjoys tower defence games. It definitely was already worth the money before the actual release and now even more so.
