Yes, the game is very dated, but once you get into the storyline, you won't even notice, as the game is about information more than anything. I'd LOVE to see an updated version of this game released someday.\t\t\t\t\t

\r\n\t\t\t\t\tI remember this game like it was yesterday. This is how I first applied to the Agency, using this game. I was one of the first recruits to be digitally recruited using the internet. In the mid-90s I was finishing up college finals and getting ready for Christmas break. I was chatting with some of the tech nerds in the basement of the campus library and one of them mentioned that he had been playing a new game in the lab. Said the integration of video with the nypertext novel features was way ahead of its time. Recommended That I see the tech help desk and borrow a copy of the floppy disks and users manuals and get the game myself. I did.

23 years later and I still haven't fully beat the game; however, I did makeit far enough to land a job with the Wizards of Langley. So glad this exists. Trip down memory lane.\t\t\t\t\t
