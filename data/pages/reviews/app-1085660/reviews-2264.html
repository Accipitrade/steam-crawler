The game plays and feels different, not in a way that is not enjoyable, but more so controls and sensitivity wise. It feels SO good to play this is crucible, in raids, really in any mode. The SSD really is felt too. If you're on the fence, seriously give it a shot. Though be warned, As I said, this game is now in the best state it's been in for some time, and as a result that state (end game activities for example) are locked behind the season pass.
