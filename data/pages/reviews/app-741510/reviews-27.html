\r\n\t\t\t\t\tI've played the game about an hour so you could count this as an initial impression.

Let's get it out of the way, the obvious comparison is Hotline Miami. HKM holds up pretty well side by side. The action is good, but the controls feel a little loose. The game relies heavily on a dodge mechanic, and a slow-mo mechanic. The dodge mechanic feels pretty good, it allows you a brief moment of 'invincibility'. The slow-mo mechanic can be used quite liberally, and when combined with the dodge mechanic allows you to pull off some pretty gnarly kill stunts.

The controls are overall good but don't have that same snappy tight feel as Hotline. You will die a lot, that's part and parcel for a game like this. Generally speaking, the game feels challenging and punishes you without making you feel stupid. There are occasional moments where you might say 'That was cheesy' or you dodge yourself into a bullet in between stringing dodges together. The graphics are pretty much WYSIWYG, don't expect to be blown away in that department. There is a lot a lot of graphical flair which sometimes makes it hard to keep track of what is happening on screen, but it's never debilitating.

If you think you would like this game you probably will. I'm not entirely convinced it's a $20 title but you'll have a fun time. I liken this game to a puzzle where killing enemies as fast and efficiently as possible is the solution. Solid fun.
