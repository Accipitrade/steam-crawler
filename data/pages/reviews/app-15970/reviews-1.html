\r\n\t\t\t\t\tBefore I begin, I'd like to point out that this game among some other Luxor titles do NOT work on Windows 10.
If you purchased this game but can't play it on Windows 10, check out this thread to fix it.
Huge thanks to Fluttershy for providing fixes for these games.

I am aware the HD versions do work on Windows 10, but since this review is about this version and not the HD version, I'll be judging this version of the game.
With that out of the way, let's check out my review of Luxor.

Intro

If you've ever been part of the sphere-matching craze like I have, there's a good chance you've heard of this game.
You match at least 3 balls of the same color to destroy them, for which you gain points.
You do that by shooting them from your shooter at the bottom of the screen, which moves along the bottom of the screen.
These balls move along a set path depending on the level to your pyramid, which you have to prevent from happening.
When the balls reach the pyramid, you lose a life.
More points are provided the more balls you destroy, with bonus points and power-ups being dropped frequently.

The balls are being pushed by a rather cute-looking scarab, and destroying all balls it pushes destroys it as well, giving you bonus points.
In order to win you need to destroy all the balls these scarabs push until no more spawn.

Personally I'm very nostalgic when I'm playing this game, since I loved these kind of games when I was younger.
It's very much like Zuma in regards of the general gameplay.
There's however distinct differences from it that set it apart from Zuma.

  • The Egyptian theme
  • The way the balls form a line (or...lines, rather)
  • The playstyle being more focused on destroying balls instead of clever tricks

Personally I'm a big fan of anything Egyptian, though I understand it's not to everyone's taste.
The way the balls form a line with the scarab system is a smart choice considering you're forced to shoot from the bottom of the screen, whereas Zuma didn't need it due to the shooter shooting on it's axis.
Incorporating bonuses for destroying a line was also a neat decision.

The game's playstyle focusing on simply destroying balls as opposed to clever tricks and going for that one satisfying combo shot is a minus to me, since I like playing it smart and making a strategy to get more points.
Instead the game punished you for 'missing a shot', which can prevent you from getting that perfect score on a level.

Price
Since I've made comparisons between this game and Zuma, I'm gonna compare their prices as well.
You can pick this game up for 9,99 on steam.
Although the price isn't shabby, it's a shame the game doesn't normally work on Windows 10 as mentioned earlier.
Compared to Zuma which is only 4,99 and works without any issue whatsoever.

Is the game as good as Zuma?
That's a personal taste, since they offer different kinds of playstyles.
Do I recommend this game?
Yes, especially if you enjoyed Zuma, though you should get it when it's on sale since the developers can't fix this version of the game to work on Windows 10.
I'd recommend you get the HD version instead since that one does work out of the box, but if you want the old graphics for nostalgia you should get this game on sale.\t\t\t\t\t
