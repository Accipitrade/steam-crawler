\r\n\t\t\t\t\tA surprisingly good game, the video doesn't do it justice. The ability to make log entries about systems allows you to make notes to keep track of all the various quests you end up going on and the modular ship design allows for enormous satisfaction when you blast other ships to pieces.

There are some criticisms I have - it would be good to have the ability to tractor some objects and lead them around to new points. It is fairly annoying having to "juggle" resources so that you can add new modules and there is no "structure" module I've found so far. If you want to build a ship with structures so that you can make interesting shapes... you'll be dissapointed. It's also not amazingly self-explanatory when it comes to navigating the galaxy - at first you feel like it's a path leading from system to system but when you don't find another connection, you assume you've explored to the full extent. That isn't true - a lot of the galaxy can be discovered -outside- of the set path and you can fly into deep space.\t\t\t\t\t

\r\n\t\t\t\t\tSo I've thought it would be something like S.P.A.Z. but it's more of an exploration/puzzle type of game.
There is fighting and shooting, but it's not the main point of the game and mostly things don't revolve around it.
The game has some nice writing overall and world building, throwing some choices at you and puzzle solving. Some puzzles I enjoyed more than the others as should be expected (some are just too vague in whatever is required of you), but beyond all I enjoyed how you will basically travel the whole world to finish the game, so while there are some filler locations, in general all of this is connected somehow to a quest or quests.
It doesn't do a very good job at explaining how much upgrades (not all) actually bring to the table, like a generic "increased range", how much range is a question you can't answer beforehand. Not that it really matters in the grand scale, but it still takes time to try different tech that you eventually unlock and learn which is useful and which is not that much.
Hint for the game - Use map markers to spot whatever weird things/planets you find. It'll save a lot of time and did so in my case. Checking the Galaxy manually in other case would be a drag.
The game is way longer than I have originally anticipated and it's a good thing.
Resources are confusing, who trades what for which resource is a mess and there is no UI hint for that until you memorize things, only Steam Guides to help you out. There are places and things you should unlock early on the map (and hopefully you will) otherwise you will be having a difficult time.

UI is lacking, but the exploration feel of the game is nice. Doubt I'll ever play it again as there is too much flying around with nothing happening, but completing this game once for the first time was a fun experience.\t\t\t\t\t
