\r\n\t\t\t\t\tSmack Studio is a bit like Rivals of Aether, but you get so much more.
I know a lot of folks played Rivals of Aether because it allows for custom characters, but creating a custom character in Rivals of Aether isn't as straight foward as it is in Smack Studio. Smack Studio comes bundled with all the tools you need to create a fully fledged custom character and the community on Discord is very helpful.

As a programmer, I could immediatly see the potential within the character editor. Not only can you make animations, hitboxes and visual effects very easily, the tool also provides advanced features in the form of code blocks named "actions" these "actions" can be triggered from any animation.

This game/software could easily be worth double or triple the price for what it offers.

There are still limitations, but the devs are working hard on bringing new features to make Smack Studio the best it can be.
Devs also take feedback very seriously and are very active on the game discord.\t\t\t\t\t

\r\n\t\t\t\t\tSo I was initially pretty skeptical of the game. I saw a Youtube short that showed it off and I remember thinking "There's no way its that simple" So I bought it and holy crap it is super simple to just make a character and go. I can't really speak much on the actual fighting aspect of the game but in terms of the character creation tool it is super well done and I am very excited to see how it improves. I have experience working in 3D modeling software so the bones and rigging wasn't very difficult for me to comprehend but there are a few things I would like to see maybe a video on in the future.

-First off I think a tutorial on making a character from scratch would be really helpful since it took me a while to get that ball rolling but again I'm experienced so it didn't take long.
-Second I still don't exactly know if there is a way to remove the black border on characters or not.
-Lastly when it comes to the testing mode I think it would be nice to have a longer stage in the demo since I want to try and see how my combos look but the dummy falls of the stage after only 2-3 hits. Either having an option for a fully flat stage or just a longer one without platforms could be helpful.

Overall I can't wait to see how this game improves and I really hope more people are able to see this because the tech in this thing is insane! And thanks to the devs for being interactive with their community here's to hoping it grows much larger for the full release.\t\t\t\t\t

\r\n\t\t\t\t\tThe game has a solid base and I expect it will develop into something more complex, beautiful, and popular. As of this review this game has 2 facets: fighting and creating, and both are strong.

The fighting is easily reminiscent of Smash Bros. Damage the enemies to knock them out of the map. There are your basic left/right, down, and up attacks, special attacks, grabs, shields, and dodges, and then there is an Ultra attack which can charge up multiple times during a fight and is a nice game changer. The 6 characters that come included with the game tease some of the archetypes you can model when you are ready to switch over to create mode.

Create mode is versatile, but if you are a beginner like me, you definitely want to watch the YouTube tutorial to get started. I am impressed by the simplicity of the animation rigging system; the character model will move seamlessly between the frames you predesignate. Depending on how many frames you place between poses will determine how quickly or slowly your character changes poses, and you have complete control on actions, sounds, and hitboxes frame-by-frame.

Overall, very excited for this game.\t\t\t\t\t
