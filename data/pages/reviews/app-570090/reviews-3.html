\r\n\t\t\t\t\t- Worth upgrading to?
An iterative upgrade rather than a full new package (as advertised). Not worth full price if you already have v2 and it does what you want.

- Worth buying as a new user?
Articy Draft doesn't do anything you can't achieve separately using apps like MS Office/Google Docs/Draw.IO, but it takes all of these disparate tools and unifies them in a reasonably intuitive package. If you want to create a book, screenplay or Tabletop RPG campaign it will be an excellent tool to aid in planning and structuring the entire project. It can also be used in a similar vein for digital games design, with features specifically oriented towards exporting of data to game engines such as Unity or in JSON. New to this version is also some degree of automation, which would be useful in game design when creating DLC for example. Whether it's worth buying depends on whether or not you already have an effective workflow for content creation, but overall I've found A:D a good application and worth the investment.\t\t\t\t\t

Unintuitive controls. This also leads to crashing, see above.

I was hoping this would help me organize my ideas for a lot of projects I have in my mind, and save me some notebooks (where I normally keep all my ideas). Instead I have a program that wants to look like it works, but in fact does nothing, crashes, and loses my information every 10 minutes. Be prepared to memorize things when you find them, because it won't tell you a second time.

I feel let down by this. It has been on my wishlist for a few years (from articy draft 2 days) and now that I have it, I regret it, and am going to be requesting a refund.

Find other options. Really.

Additionally, support is moderate (though not great or awful) and the tools are fairly generic in their capabilities. I get the impression they're more selling flash and dreams than actual valuable software intended for reliably and easily getting work done.

The UI remains largely the same from version 2 without significant productivity enhancing features, and the price remains high for what it is.

\r\n\t\t\t\t\tArticy draft 3 (AD3) is the answer to my prayers as I've been searching high and low for a "narrative-mapping" software. As a game developer, I found AD3 very helpful in many aspects. It can help you with visualizing the flow of your story and narratives, be it linear or non-linear.

AD3 also allows you to store all the details of your characters, items, NPCs etc. On top of that, another very useful feature of AD3 is that you can use it to plan any location/map in your game. For instance, you can overlay a zone / patrol path / landmarks / spots etc. on top of your location/map screenshot.

In conclusion, this is a very good software for developers / storywriters, even at full price.\t\t\t\t\t

\r\n\t\t\t\t\tStill early stage of using Articy draft 3 but been using v2 for a while but here's my review.
There are two types of people who may be interested in this, v1/v2 people and new to the program, so I'll speak to both parties.

New users
Articy draft is one of the best programs for writers, designers and developers. It DOES NOT make games but its designed to hold all the information on the narrative of a game or story/novel. If you need a platform to not only write a story but to also formulate plotline and create branching conversations, while keeping track of who is where doing whatever, then this is an excellent resource.

I mostly use it for writing (novel and short stories) and the ability to organize my characters and there traits as well as keep track of everything they have said is a god send. Creating my own templates for my characters makes remembering what a person looks and acts like a Breese. These are just a few of the parts I use and it has too many for a review here.

If you want/are designing a game or story that has a number of characters and plot points then this can easily be a program you will find indispensable. So for all the writers and developers out there I highly recommend.

V1/V2 users
You know how good the program is already so I don't need to sell you there. This is about new features. It looks the same and works the same. The differences between v2 and v3 are fairly small but very useful. Personally I use the documents section a lot and the added features there help a lot. If you use unity game engine though then upgrading to v3 is a very good idea the export feature I would imagine will save a lot of time and effort. My recommendation is that if you are fairly happy with v2 and the new features are not something that shouts "I have to have this" then you may be happy sticking with v2 for a while longer. However if your still using v1 then this is defiantly a recommended upgrade. The number of upgraded and better features makes the expense very much worth it.\t\t\t\t\t
