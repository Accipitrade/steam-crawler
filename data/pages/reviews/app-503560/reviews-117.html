\r\n\t\t\t\t\t911 Operator is a game with a lot of potential, but it's not living up to that potential. Menus are easy to navigate, and gameplay is simple, yet fun. Since the game is so simple, it can get pretty boring once you hit 5 hours of game time. In the game, you play as a 911 operator that dispatches different first responders like police, fire fighters, and medics. You also answer 911 calls, which is probably the highlight of the game.

The problem this game has is its variety. There are only so many different emergency scenarios and 911 calls in the game, and it doesn't take that long before you've seen all the emergencies. Also the scenarios repeat themselves, so you can get the same exact 911 call five times in one game. There is a game mode that doesn't have them repeat, but it shouldn't be a game mode that fixes that, it should be fixed in the entire game.

When I hear the same person complain about a bat in their apartment five different times, I can get pretty annoyed, and it completely breaks immersion. A plus to this game is that you can pick any city or small town you want that's in the U.S., so that helps with replayability. For the current asking price, this game is not worth it at all. It's good if you like management sims, but that's just about all you're gonna get from this game.

I do recommend this game, only because I find it fun to play every once in a while. I would suggest only get this game if it's on sale though. Current asking price is 15 dollars, but this game can go on sale for as low as 1 dollar. Either way you look at it, this is a pretty solid game, just with some issues.\t\t\t\t\t

\r\n\t\t\t\t\tI really wanted to like this game and I think there is a framework for a fun experience here, but I got bored 90 minutes in and gave up. It feels like the game just requires you to complete the same tasks that don't really change much outside of needing more units to be more efficient during the game. The interface is clunky when trying to manage multiple units in close proximity and it's pointless to be able to select one in the list if it won't center on the unit or show you where it is before you assign it to move to a location.

The developer has gone to the trouble of putting some cool calls together and for that, they should be commended. But as a whole, the game feels like it is still in Early Access.

Edit: I reinstalled the game to see if I had given it a fair shake as I had only played it briefly before and after getting to Albuquerque I can say that I cannot recommend this game less. The actions are repetitive even if the dialogue is interesting and done well. Hopefully your game won't get stuck with several police actions to complete and no police unit actually completing prisoner transports. I had to ignore the events to get the day to end. I'm honestly sad that I bought this, even on sale.\t\t\t\t\t

Bar this one tiny downside, it's a very addictive game that lets you play any location in the world.
