\r\n\t\t\t\t\tAt the time of this review I played around 47 hours of this game.

I came to this game with a lot of expectations since it's supposed to, "mimic" the role of a 911 dispatcher, and I was not disappointed. When originally starting off at the tutorial, I couldn't stop playing and couldn't stop getting stressed out. Then I discovered the other game modes and I was immediately hooked on.

911 Operator has an excellent game design that can't be replicated to exactly copy what real life police dispatchers do, but reasonably does so in a way that can be very fun, enjoyable, and equally frustrating.

The game is strong in it's entirety, but each components that makes up the game has a fair amount of strong weaknesses which I am listing below.

1. Repetitive Calls
Now I get the game is to be responding to emergency calls and of course there's going to be plenty of the same calls. However if you play the game for a decent amount of time you start to notice the same calls from the same voice actors over and over again, and the dialog options can be remembered to where it begins to feel dull and annoying. Multiple varieties and different actors for emergencies can definitely solve this, as there is only a couple variations with the same people voicing emergencies.

2. User Interface
The user interface doesn't have too many negatives and shows very cleanly, assuming you are not going on hardened/expert difficulties. However when you are confronted with these difficulties and reap the rewards of high payouts, you have more units that continue to clunk the interface. This is a problem as sometimes when directing units I have a hard time determining which unit is on the way to a specific incident when the routes are all the same color. Granted however, you can click a unit and the event will make an indication, but still have great difficulty when managing multiple things at once. There is also the movable box where you can read the text of a call, or look at the units at a scene. I have personally clicked that box so many times it moves into the middle or off my screen that I need a second to put it back into place, during which may cause problems.

3. Lack of unique situations/environments
Unfortunately this game doesn't have plenty of unique situations. While there are some that definitely add spice to the game such as the Riots, Music Festival, and Fireworks, they feel too repetitive as well. There's usually only one way to go about these incidents and they are the exact same way every time no matter what map or duties you are doing. There should be a lot more variety that not only has plentiful options for variety, but also limits the operator's ability to dispatch units effectively.

Overall the negative portions can be addressed in the upcoming game, "112 Operator" so I'm psyched for that, but it turns down the replay value of the game and makes it more difficult.

I highly suggest giving this game a try and a few hours of gameplay, and those hours will be enjoyable.
