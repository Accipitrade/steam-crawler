\r\n\t\t\t\t\tAn interesting game that gets old pretty quickly. The best part of the game is that you can play it in the town or city where you live or grew up (though not everywhere is available) and seeing it on the streets you recognize makes it more interesting. Overall, it's a fun game of dealing with emergencies through the 911 system. The downside is that there aren't all that many different emergencies and you can pretty much use the same "right" response every time you get the same emergency and end up with the same result during the calls. If the game had probably about 5x the number of emergencies and there wasn't always a "right" response, the game would have far more replay value. As it is, you'll probably be bored of it within 5-10 hours as it just gets too repetitive.

The game is worth it at 75% off, but I'd avoid paying full price. There are better options out there for emergency management style games even though I can't think of any that have you playing the 911 operator.\t\t\t\t\t

The gameplay loop is fun, while the game isn’t personally for me, I would recommend this game for most people. Would I recommend this game? Yes, yes I would. Though, if I were you, I would pick this game up on sale.\t\t\t\t\t
