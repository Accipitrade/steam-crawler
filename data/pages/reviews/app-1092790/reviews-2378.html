\r\n\t\t\t\t\tAn honest review without any spoilers.

The game is divided in 3 parts. It only has a story mode. Expect to be immersed into the unique universe of Inscryption. Its best to go in blind, don't look up any guides or game play aside from the store previews. Trust me on this, you'll enjoy the game that much better. It is a card game with elements of puzzle solving, some critical thinking and board game type elements. Expect to be mind boomed, the game breaks the 4th wall, and i don't really have any complaints aside from the 2nd area (it was too RNG for me) and i liked the 1st and 3rd area best personally. If you like card games, the game is for you. If you like board games, the game is for you. If you like story driven games, the game is for you. If you like puzzles, the game is for you. If you like games that make you feel like it capture some place in your heart after you finish it, this game is for you.
