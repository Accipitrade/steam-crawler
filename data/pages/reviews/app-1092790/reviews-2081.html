\r\n\t\t\t\t\tInscryption is genuinely incredible. This game is a stunning love letter to all TCGs, CCGs, and LCGs; any fan of card games NEEDS to play this. There are countless references to classic card games, showing just how much this game is a labor of love for the genre. However, the game itself is so engaging both for it's gripping narrative, fun mechanics, escape-room discovery and exploration, as well as even ARG elements for the more dedicated fans (not necessary at all to do the ARG elements to get the full experience). This game is one I would legitimately recommend to anyone, and you REALLY should go in to this game as blind as possible.

The only thing I wish, is that there's an update with more content (SPOILERS) I would play this game forever if the cycle of Scrybes was to some degree random each time after the first playthrough, like a rogue-lite. I wouldn't need any more narrative or extra rogue-like elements on top either, the narrative already encompasses this cyclical game state so I would just love to be able to play through Act 1 and 3 with a random scrybe for each. Act 2 wouldn't even need to change!

Anyway, spoilers aside, when the only critique I have of this game is that I want MORE of it, and it was easily one of the most unique video game experiences I have ever had. Easily one of my favorite games of all time.\t\t\t\t\t

\r\n\t\t\t\t\tI think the game is great and really lives in a niche of content you can't find in a lot of places nowadays, but I don't think this game is meant for everyone. There are some things that definitely somethings that I feel are worth mentioning before someone makes a purchase.

Firstly, this game is psychological horror, I didn't even believe it at first since I didn't react to the game as I do to most horror games. But after I beat the game, I realised how much the game affected as I felt physically ill after the playthrough.

This game is meant to be a thinking game, and one that isn't meant to really reward you for it. This game has a lot of puzzles, so much so that completing a puzzle usually only gives a harder puzzle as a reward. Now if you like puzzles in games, then this is great, you're piecing together a story, and trying to discover the true narrative of events, if you don't like puzzles, this game might be a pass, for purchase, since the story is pretty well documented in playthroughs, guides, and in the wiki.

Regardless of those two notions, I think this game is wonderful and is worth the money, time and effort.

\r\n\t\t\t\t\tReally cool game. You have to buy this game if you are passionate about card games and the culture that surrounds them, because the developers sure were passionate about it. This is not only a love-letter to card games, it's also a love-letter to creepy urban legends. I won't give away too much, because there are a lot of twists and turns. Suffice to say, this game is a mind-bending experience and it's exhilarating to watch the story unfold as you progress. I think some of the ARG stuff was a bit blunt and cringy for my taste, but it's all good because everything else makes up for it. Beautiful soundtrack, great art, atmosphere out the ass... This game has so much going for it.

Now, let me talk about the actual meat of the game: the card game. Without spoiling too much, there are a number of different card games with their unique mechanics in this game. The encounters can be quite challenging, but nothing too crazy difficult. The games are not balanced well, which is why they're so amazing. These games have so many fun combos that are absolutely broken when exploited. These games were meant to be broken, which is fine since this is a single-player game. You have to come up with broken and unfair combos to beat the tougher encounters. It's so gratifying to come up with a ridiculously broken strategy, and the game actively encourages you to exploit all the crazy possibilities. The different card games provide a refreshing variety to the overall game, and they also showcase different types of card game mechanics.

Great game, you gotta play this, so much ♥♥♥♥♥♥♥ fun. Even if you aren't into card games, I still think you can get a lot out of this thing.\t\t\t\t\t

If you think like a games journalist, Inscryption is the Dark Souls of puzzle games. It is going to wreck you with brand new rules you had no way to understand, repeatedly, like a cheating homebrew DM, who then proceeds to kill you and steal your soul for losing. Unlike previous games like Pony Island, Inscryption is actually quite long and leans very heavily on the base-level game--the cardgame--so you actually do need to be into that if you want to engage with the game for its full length. Luckily, it's actually pretty good and also very breakable, once you figure out all the rules. The first-order metagame also provides several ways to make the core game even more broken, to help you through the roguelike elements.\t\t\t\t\t
