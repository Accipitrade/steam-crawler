\r\n\t\t\t\t\tIt's difficult to describe just how amazing this game is, but I cannot urge you enough not to spoil yourself at all before playing. This game is the embodiment of "Subversion of expectation", and if you're going into this totally blind I can guarantee that you will have an experience like no other. The game will teach you basic principles in a fun and engaging way, then promptly flip everything on it's head while keeping the skills you learned applicable. I am yet to find another game that pulls off this kind of gameplay switch as effectively and intuitively as Inscryption.
In terms of visuals the game magnificently mimics the style of late 90s / early 2000s PC games and not just as a gimmick, but makes effective use of aspects like the simple lighting and limited palette to shroud things that are not supposed to be seen in pure blackness in a way that feels fitting.
All things considered I think this game is a must-play, even (or perhaps especially) if you believe that you don't like card games, as I did before playing.\t\t\t\t\t

\r\n\t\t\t\t\tThis game was absolutely captivating from start to finish!

I won't let there be spoilers for the game but for me, having a first glimpse at the game It absolutely intrigued me with having the tags of being a card game but having these psychological horror aspects within it's environment. After seeing it I knew I had to try it out for myself and completely didn't expect there to be such lore at all going into it. From the chilling environment to just the strategic game play of this card game, it was so fun and so enjoyable to play. Totally would recommend just going into it completely blind, don't watch playthroughs of this if you really want to have a full experience with Inscryption. There was only really one mechanic within the game that turned me off a bit which would be a bit of a spoiler so I won't dive into it but aside from one minor mechanic, this game is fantastic!

You definitely need to be able to focus and be able to use strategy with this game to be able to progress through what this story holds but as you learn, the stronger and more knowledgeable you get!\t\t\t\t\t
