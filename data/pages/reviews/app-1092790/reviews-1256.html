\r\n\t\t\t\t\tBIG DISCLAIMER:

Don't buy this if what you're looking for is a roguelike card builder. As that, it is really disappointing. It has a mode that *sort of* follows that fantasy but it is clearly not what the game was made for.

This is a somewhat hard recommend, cause I feel like this is on the very edge of post modern deconstructor fleet ♥♥♥♥♥♥♥♥, but I will recommend it for the artistry in display, and somewhat fun gameplay.

and I say somewhat, because this is a game of dizziyng highs and terrifying lows. I don't know how to describe it other than "damn if this was a roguelike card battler I would like it better".

Play it if you like internet discussion on obscure stuff. But don't go in expecting Slay the Spire.\t\t\t\t\t
