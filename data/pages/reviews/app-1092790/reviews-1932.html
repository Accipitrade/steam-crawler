\r\n\t\t\t\t\tJust finished the game and I really enjoy it! The game has a lot of surprises and I feel that talking too much about the game here will spoil the surprises for you. I would recommend this game to people who loves mysteries and puzzles.

Although the core of the game is a card game, I would say it is more akin to a puzzle game where you're given certain rules and resources, and try to solve the puzzle with these. As for the horror part, there is not much of horror but some creepy and eerie moments that make you go "WTF".

If you still want to know more about the gameplay as you're not sure about the whole card game thingy, you can read more below which is spoiler-free but it might ruin some of the surprises for you:
---------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------
Game is break into 3 Acts, the gameplay is still card game but the rules of the game changes with each Acts.
Act 1 - Roguelite deck builder
Act 2 - Constructed deck (You build a 20 cards deck with the pool of cards, which you can get from winning battles or opening booster packs)
Act 3 - Roguelite deck builder with more complex rules

It is also a puzzle game where you can interact with the environment by getting away from the "gameboard / table". There are some 4th wall-breaking / unique player-game interactions.

Story: The game tells enough of story but still have some missing pieces for you to solve, and you can find out more about the secrets and story by participating the Augmented Reality Game (ARG) after finishing the game (similar to The Secret World). The ARG has been solved by the community already but it is still fun to read how the big brains manage to solve the huge and complex puzzles.\t\t\t\t\t
