\r\n\t\t\t\t\tPretty interesting game, the atmosphere is very comfy early on, the card game keeps changing wildly until the end. There's a bit of meta-narrative which I don't always care for but it was fine in this and never really got in the way of the game.
The card game itself I think was at its best in the earlier parts of the game, eventually so many mechanics are introduced (or dropped) that the game becomes very different.
The one part of this game I didn't really care for was
One of the bosses opens a real file viewer and demands you select a file from your computer, you're rewarded based on the age of the file as part of the mechanics of the bossfight. Whichever file you choose, the game then threatens to delete this file if something specific happens during the fight.
I don't believe it can actually delete it but I don't like this kind of 4th wall breaking very much.
Aside from that my only complaint is that there wasn't more to play with the necromancer and the wizard, and the game even lampshades this towards the end.

Update: With the kaycee's mod additon, which turns the beginning part of the game into a roguelite, I've been playing this game a lot more recently, adds very nice replay value.\t\t\t\t\t

\r\n\t\t\t\t\tInscryption would be best described as a card game with horror elements. It’s not really a horror game per se, it has those elements and decently put together story but horror isn’t the main drive here. What it doesn’t do in horror it makes up for in story, core gameplay, and worldbuilding.


Gameplay

Inscryption is a card game, it uses lanes in which creatures fight each other and deal damage to the opponent. One aspect of the game that I love is that players don’t have health per se, more like there is a balance of power, where there is a set of scales, sort of like a reverse tug of war. When a player brings down the scale on the player, that player loses and therefore the other one wins.

The core game is very tactical and very fast, one thing that is fantastic about the game is that there is no thinking time from the computer. To be fair, it plays a certain deck and sticks solidly to a set strategy. However the deckbuilding aspect of the game is far more like Slay the Spire, even the map in game is very much based around that. You have to collect creatures, improve them, alter them to create some truly bizarre and powerful combos. When you know what you are doing, the combat can be a range of easy to challenging. Collecting the cards and making the deck streamlined is essential and the mechanisms to do that are fantastic, better than Slay the Spire even. The game has so many small tools and strats that it feels like a very varied experience.

In terms of the horror gameplay, it’s something I am going to be intentionally blurry over so as not to spoil anything. The interface is very point and click, simple and effective. The game rewards close inspection of the environment and not taking your time will punish you. One thing I will say is that there is a bit of a ‘fat finger’ feel with the cursor, it’s easy to flub and interface because as you’re busy skipping text it’s quite possible that you would activate something you didn’t mean to. Also one important note, the gameplay is not… consistent.

In terms of core gameplay the game is good, however it is easy to reach the limits and you might be wanting more at times. Also it can get a little repetitive but the game is constantly breadcrumbing new content so you never feel as if the game is grindy. There is a sharpness to everything, stuff moves fast and feels precise. The feedback from the interface and environment is excellent and you get a lot of cues about things in several places.


Aesthetics

I wouldn’t say the game is beautiful, it is very well polished but it isn’t as striking as Darkest Dungeon or as slick as Doom Eternal. It’s a little rough but it does serve it all with a menacing atmosphere but it’s more overly unsettling. As stated, it’s a card game first not a horror game, where Daniel Mullins created Pony Island which was a beautiful mash of both genres, this game is more about the gameplay.
To be fair to the game, I cannot give examples or state too much. What you might need to keep in mind is that the game aesthetics are not consistent. However overall they are functional and serve to keep the player on their toes.

The story is excellent as well, throughout the game my most used expression was ‘what the ♥♥♥♥?’. Not used in a scared sort of way, but uncovering something that changed the perspective and scope of the game that threw what I had mentally built out the window. Warning, this game has a massive rabbit hole to go down, and the easter eggs only get more insidious and obscure. The characters are also interesting as well and have long histories and interactions which again throw everything up in the air at times.


Overall

What you have to keep in mind of this review is how much had to be redacted. The horror genre is difficult to review for that reason because a lot of generalities have to be used. That said, the game overall decent in terms of gameplay and aesthetics. It’s not going to be life changing, however it is going to be a good couple of nights worth of entertainment. One downside to the game is how short it is, to be fair, the amount of content in that time is excellent. The variety of content is probably the game’s strongest feature and part of what is unsettling about it is how much it can change. What I would say that this game will easily have you playing a second time, there are a lot of secrets, some of which are incredibly obscure, there is a whole Reddit megathread dedicated to this.

Now the game retails at £16.79, for an indie game like this that is a fair price. Let me put it this way, buying this game at full price is worth it. If the game ever went on sale for somewhere near £10 this would be a must buy. In the long term this would be a great game for a rainy day. You might have played the game already but on another playthrough you might find something new, play a certain strategy or simply enjoy the ride. The horror in this game would be great for kids, it’s about that level. It can still give you the heebie jeebies as it goes off course but it’s not visceral or abusive.

Inscryption made it to my mid 90s on my top 100 list. It’s not a standout game for all time but it’s a very well-built game overall, it’s going to be a game people point to when comparing it to other games as what Daniel Mullins got right and other games fall down on.
