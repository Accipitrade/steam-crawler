\r\n\t\t\t\t\tThe game is overall very enjoyable whether you want to play the story or you want the rogue-lite experience and just go straight to kaycee's mod. My one big gripe with the game is the restart penalty when you prematurely restart a run in kaycee's mod. The game puts you at a hard disadvantage for restarting a run in a game where doing multiple runs is the point. I understand why they do it, so that you can't just continually restart to get the "god" run. While it does work in that regard it also can make kaycee's mod feel very tedious as sometimes the game will give you an essntially un-winnable start that you're either forced to throw, so you can restart penalty free which takes more time than it should, or restart the run early and take the penalty. It just seems that there should be a better system to deter the player from restarting that isn't so cruel in a game where getting in run after run is the point.

Overall, the game is very enjoyable if deckbuilders are your thing. If you don't like deckbuilders or rouge-likes, I would not suggest this game.\t\t\t\t\t
