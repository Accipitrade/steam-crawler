Is the game good?
Yes, it is. Completely worth the money.

What kind of game is this?
A rogue-like card game with tabletop elements.

About the game:
It can be divided in 3 or 4 essential parts:
1 - That's the playable part of the demo, but it is expanded; 3 main bosses (besides the last one); many upgrades that you can get progressing in the board; escape room; first person.

2 - This is the part that everyone (or almost) got surprised. The game starts looking like an old Pokemon game, but it still is a cardgame, which will evolve as you progress, adding new cards (you get them by opening card packs, that you obtain by exploring each of the 4 zones) and new systems; 4 main bosses, that you need to defeat in order to end this part; as it looks like a Pokemon game, the camera will be above you.
If you don't like this style, you can simply rush it, since the game will get back to the style of the first part.

3 - Almost equal to the first part: scape room; first person; rogue-like; tabletop elements; 4 main bosses; different board, with other upgrades; new cards and some from part 2; new systems from part 2 are back, but different.
ADVICE: look in all 4 directions, you will need the 5 pelts if you want to know more about the game's story.

4 - This part is just the ending, can even be consided as part 3, but it is a special part of the game. Works just like part 1 and 3, but is a """boss rush""".

Personal opinion: This game is good, but it's missing a multiplayer mode, one that they would need to change the game's entire system to work as a fair game. They should've told us about what happens in the second part, or at least say that it changes (almost dropped the game thinking that this would be the rest of it).
