\r\n\t\t\t\t\tThis game is insane. It's heavily polished, has an incredible visual style, and also does an amazing job at distilling what could have been a mess of menu systems into a completely diegetic, simple and intuitive interface.

Upfront the game establishes very little rules, making you fear what assumptions of yours the game will break next. And boy does the game do this a lot. My jaw hit the table numerous times over the course of the game in awe.

My only real complaint: if you are going to mess with the structure of the game (to the extent that this game does), make sure you give the player hints about how long until the game gets "good again". The game looses a lot of the 'juice' that really made it shine in the first act, and there were no hints about how long it would last. I almost thought that the game made a permanent downgrade to itself and considered quitting.

Also if you are not a hardcore deck-building card game player, it is ESSENTIAL to use the 'Auto-complete' button in act two. It actually works really well, and the game makes no hints about using it. Ugh.\t\t\t\t\t
