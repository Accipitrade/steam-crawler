\r\n\t\t\t\t\tThis is a very good game.
However, don't be fooled, this isn't a serious card / deck building game. It is a story game that uses a card game as part of the plot. Experienced deck builder fans will find this game very easy, as it is very easy to exploit the game to create extremely overpowered decks. This was done intentionally to help unexperienced players progress through the plot.

After you beat the game, there is a more balanced, although still exploitable, version of the game called Kaycee's mod that is unlocked.

Some positives:
- The game has a very good atmosphere and the visuals are great
- The card game is fun, even though it isn't too hard
- The story is engaging
- The characters are interesting and well written
- Kaycee's mod is a lot of fun and adds reliability
Cons:
- The card game is easy
- The second and third acts of the game aren't nearly as good as the first (although I still enjoyed them)
- The main game is short

Overall rating: 8/10\t\t\t\t\t
