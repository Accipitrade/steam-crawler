this addition is great. The main game is good. And if you want to play a roguelike deckbuilder: well it's now part of the game. and it's an entertaining part of the game.

-- previous review--
it is not a "roguelite deck builder with a meta story and progression on top of it."
that is just one part of the game (about 1/3 of the total game time)

the game is mostly the story that switches between different card games that share some core mechanics. (and the art-style changes a lot). the first third has some replayability, but the rest only if you like secret hunting imo.
don't buy if you expect a roguelite. You might enjoy it if you enjoy the other games Daniel mullins made / if you enjoy meta level twists\t\t\t\t\t
