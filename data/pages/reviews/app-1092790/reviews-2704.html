\r\n\t\t\t\t\tI absolutely adored this game and it leaves it very open for future content, if it does

The first part of the game is everything it says it is on the tin. A roguelite, horror, escape-room esk game. While all the cards are fairly basic, wolves and bears and bugs, it does all of these things very well. Further, while there isn't a ton of things to run around and collect from the cabin (least as far as I could tell), it is still fun solving the puzzles and attempting to open and unlock all the shiny bits. If I were to complain about anything, its very easy to end up with very game breaking cards or mechanics. Everything about the first half of the game screams that it wants you to have as much fun as you can while being a horror game. Its also deceptively short.

The second and third parts of the game are a completely different beast entirely. As to avoid spoilers, as I think going into this game as blind as possible is the best way to experience this and any other of Daniel Mullins games, I'll keep it as vague as possible. What you see in the first part of the game is largely scrapped once you reach the second half, and only slightly brought back for the third part. Do not go into this game expecting it to be ten, twenty hours of what you see in the screenshots you will be very disappointed. That said, the game becomes wildly more complex upon the second and third parts. It also no longer is a roguelite, while still having randomized mechanics similar to part one. The story is also phenomenal. There are so many twists, turns, and in general tons of fun to be had. If you love card games, creepy stories, plenty of secrets, and don't mind twisting rules, mechanics, and general gameplay, you will have an absolutely amazing time with this game.\t\t\t\t\t

\r\n\t\t\t\t\tWhen I first saw this game, I was very intrigued. I first saw this as "sacrifices must be made" or something along the lines of that title, and have been waiting for this to release since. I must say, this is what I had hoped for and so much more. The base gameplay is extremely enjoyable, and while it does get a little rough when they start implementing new cards and eventually comepletely changing the game style on you, I think its worth it for the story. So far, this title has tied in very well with the same motif as the dev's previous work, "the Hex," and if you haven't played that game I would personally tell you to check it out if you enjoyed this game.

Do NOT go into this game assuming that it is just a deckbuilding game, It is so much more and if you want a game that is entirely based around cards / deckbuilding, this game is not for you. That being said, this is a game that you will (probably) love if you love creepypastas, ARG's or anything on that side of the internet. That's all I'm gonna say about that, so if a story rich game with some deck building elements sounds like a fun time (which I think it is), then I would say you would absolutely enjoy this game.

(Note that I HAVE NOT finnished this game so idk if the ending is good or not lmao, so if the ending ends up being lacking I apologize)\t\t\t\t\t

\r\n\t\t\t\t\tI only played for a few hours, enough to get me through a victory through a run. I don't really know what this game has in store past the first win, but I'm excited to find out. This game captivated me almost instantly. The atmosphere is amazingly well done, the story is interesting and mysterious, enough to hook me in, and the gameplay is top notch.

Only 3 hours in and it's already one of my favorite roguelikes. Even if the game actually ended with the first victory, and from there it was just playing the card game, I'd be extremely satisfied. But there's so obviously more to come. Can't wait. Will maybe update this review when I've seen as much as I can.

UPDATE: After beating the final boss and restarting to what I thought would be a new run, the game instead switched genres and art style. While I love the card mechanics, they don't completely carry over to the new genre. Instead of this game playing like a roguelike, the game now feels more like an RPG. I cannot recommend this game as I don't think the card mechanics work nearly as well without the items and squirrel deck that you get in the roguelike portion. In addition...it's not described anywhere to change, thus I can't recommend this.\t\t\t\t\t
