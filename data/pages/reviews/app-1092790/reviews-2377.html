\r\n\t\t\t\t\tProbably one of the best puzzle games of all time. Kind of reminds me of old-school Yu-Gi-Oh! where the rules weren't very clear and you could make stuff up on the spot. It has a good story, with some ARG elements. This is one of those games that are so well written, you actually start to care about the characters in it. The devs do a lot of cool and creative stuff with the game so almost every fight is interesting with some twist, and the twists only get better the more you progress.

I enjoyed grinding stuff out and make a really OP deck. I wish the game was a bit longer or had a game mode where you get to use your OP deck more often.Would love to see a longer version of this game with more grinding, like maybe beat x number of bosses to unlock more ARG secrets or unlock new cards/upgrades. Not to say that the game itself is short, but you can probably beat it in a couple hours or so.

While I would love to see this game in multiplayer, I don't think it would work, just because of how easily it for you to make a broken deck. Maybe it can work with some nerfs and balances.

10/10 would recommend this game if you like puzzles or card games.\t\t\t\t\t
