This gameplay however, is not the highlight, the story is. This results in a game with less replayability and overall completion time.

\r\n\t\t\t\t\tFirst part of the game is outstanding. It's well crafted, fun to play, very moody. Sadly, I can't say any of that about the rest of the game. For some reason it feels like after finishing the first part, developers decided to do something completely different, only loosely connected to it. The game is no longer a roguelike, it's no longer escape the room space, it's no longer moody horror game and becomes sort of surrealistic experience instead. The storytelling becomes quite mediocre, and the plot hardly goes much deeper than "there's 4 guys, one of them is going nuts". The sad part, is that I basically came up with at least 1 better story to build the game around while playing it.

Would I play it if I knew it from the begginging? Most likely, yes, because I like creative writing, and this game is quite creative. But I also feel cheated for how different the game actually ended up to be in part 2 and 3 from the impression it made in part 1.\t\t\t\t\t

\r\n\t\t\t\t\tTHE DEMO IS BARELY ACCURATE

This game is not a roguelike/roguelite. It is a story game.

If you want a story this is a good game. if you want a mechanically fun card game, look somewhere else

The first act of this game is good fun, interesting mechanics, mysterious, and a fun to play card game with some roguelike elements. And then all that gets launched into the Bootes void. and is replaced by some very boring dragged out and predictable story. Act three has some kinda cool boss mechanics, but thats not enough too save the game from its mechanical blunders.

If you are familiar with Daniel Mullins games and enjoy them, then you will likely enjoy this game as well. However calling this game a roguelike, strategy, or puzzle game is just a lie. Only 1 third of the game can barely be considered a roguelike. while there is room for strategy, it doesnt matter, you can blindly choose whatever you feel like and will have no issues. As for the puzzle, maybe you could call some of them puzzles, but for the most part all the answers are handed too you on a platter. Maybe if you want to call the meta elements of the game a puzzle i can roll with that but once all that is solved its over, the game wont provide you with a puzzle experience.

Under the "Daniel Mullins" there is a decent base for what could be a really cool game. Unfortunately is buried under "Stuff".

Act 1 - 8/10
Act 2 - 0/10
Act 3 - 3/10

If you want a game, dont buy this.
If you want a story, go for it.\t\t\t\t\t
