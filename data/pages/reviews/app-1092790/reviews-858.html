\r\n\t\t\t\t\tInscrpytion is a rogue like card game where previous runs impact the current run. Each run potentially unlocks different card and mechanics.

The game itself has incredible flavor and atmosphere. All of the cards and mechanics feel like they've been very strategically chosen to impart that same sense of foreboding, creepiness and doom. The game makes you feel claustrophobic and disoriented through creative usage of lighting, soundtrack, font. It also breaks the 4th wall in creative ways. The game is a true Artistic masterpiece.

However, the main issue with the game is...the game. The mechanics for the each run can get pretty stale. The gameplay is not particularly innovative, and generating overpowered decks doesn't seem to be too difficult. It can become a slog to get through it, as a means to understand the plot or enjoy new mechanics the game has to offer.

Kaycee's Mod is an additional mode where it only takes the roguelike elements of the game and places it together. I haven't tried it yet, and, honestly, I'm not that interested in it.\t\t\t\t\t
