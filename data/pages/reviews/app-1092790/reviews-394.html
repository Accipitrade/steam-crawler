\r\n\t\t\t\t\tHuge fan of deck building games. Also really enjoy RPGs. Saw the reviews and thought this game would be worth the buy. Couldn't have been more wrong. The game itself is easy enough to understand. There's just little fun to be had. The game feels like its designed just to make you lose which wouldn't be all that frustrating if it weren't for how they do it. RNG. It's all RNG. Everything. What you draw, What the opponent has down at the start of a match, What options you have available on the map, etc. You might go wow there must be so much replay value and you're probably right. Good luck getting there though. Not able to start new games without dying, all the UI outside of the game itself is horrid. Not ability to create the deck you want. All just RNG based garbage, 4/10 Glad I got my money back
