\r\n\t\t\t\t\tI wanted to start this off by saying I'm not really into card games that much but Inscryption blew me away and I want more of it!

For starters, buy this game. No questions asked. Sale or no sale, this game is amazing and for a myriad of reasons.

Firstly, the content, this game is jam packed with things to do spread across 3 different acts, all revolving around the same basic concepts of winning but with twists and turns at each act. From magic to circuitry and rituals, this game has it all.

Secondly, the story of this game is something that I can only compare to a game like "The Stanley Parable" in meta levels. The game bridges the gap between the digital and real world as we spectate our lead role playing the game we are playing now. It's filled with many twists and turns and actually made me feel really bad for a lot of the people in this game. Also, this game had an entire ARG attached to it which was crazy enough.

Finally, the characters and world of this game is really creative with the way it sets itself up leading into each act. You really get a feel for each character you oppose through the game. From the moment you hit continue, the game already makes you get a character portrait of the person sitting right across from you and the game does that for every single character.

So like I said before, please buy this game if you want. It's an amazing find and nothing like I've seen before from a card game. Daniel Mullens, I salute your craftsmanship of this game and Devolver Digital, thank you for publishing it.\t\t\t\t\t

\r\n\t\t\t\t\tThis is a good game, worth every cent. My only negative comment about this game is the ending.

Gameplay is a card builder, the story has two layers each is fun and cool. The visuals are the best part for me, with a good deal of different styles and it is truly amazing the skill put into this game.

The story is a little questionable and the first half is the best part of the game, the beast half. After a certain pint the game's theme changes and so does the gameplay, though unlike the first halfs this gameplay wasn't as fun to play. The ending was also very anti-climatic, I feel the build up was stronger than the conclusion, and that the devs bit off more than they can chew with the story.

Overall a fun game but not one that you should get if you are looking for an amazing story.\t\t\t\t\t

\r\n\t\t\t\t\tInscryption is made by Daniel Mullins, a developer previously known for genre-twisting games such as Pony Island and The Hex. Most negative reviews claim this game to be misleading, but if you watch the main trailer (or know anything about the developer), it should be clear that this game will not be straightforward. That said, most of the game does focus around card-playing as well as puzzle solving.

So far, I've loved all of the games made by this developer, so again I recommend to go into this as blindly as you can, but only under the pretenses that (1) you're fine with card-playing, and (2) you'll be fine if the game switches things up on you. Even if you're not the biggest card-playing fan, just take a glance and see if you're intrigued.

This was one of the most unique video game experiences I've had in the past few years.\t\t\t\t\t

\r\n\t\t\t\t\tUsually I'm the type of person to make a funny joke about it being good, or something like that. But not this time.

This game, in my probably wrong opinion, is such a great game that its one of the few games I've modded, just to keep playing. A lot of these reviews say to go in blind, and as someone who was spoiled, Please go in blind. The game play, in my opinion, is great if you're into deck builders. However, the story is objectively great. I wont go into details, but for the love of god, don't be spoiled on it. This game was nominated for the game awards in 2 categories for a reason.

Now, if your into game play, this is normally a 12 hour game for the story, but it can be replayed. without details, there is a part that can be constantly replayed. And if that gets boring, there is an extensive modding community that are changing the game, from adding new cards, to major changes that I wont say, lest you be spoiled. But the community's strides to make the game as fun and varied as possible is shown very much.

Overall, 7.8 out of 10. To much fun and great story.\t\t\t\t\t
