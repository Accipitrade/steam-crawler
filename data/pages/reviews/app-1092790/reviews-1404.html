\r\n\t\t\t\t\tThis is a weird time we're living in. This is a meta game. This is an adventure game about card games, full of easter eggs, real world references and an actual card game gameplay mechanic. This is not what it seems in so many levels, i can't even begin describing what is wrong and what is right about this game without giving spoilers.

Totally as a game is definitely a must see. This is a sample of a next generation of games to come. The narrative is really good and the actual game, the card game itself is ok, not too hard, not too simple, more like a means to an end so even one that is not super into card games can play it. NOT A CARD GAME whatsoever. Now the confusing amazing mindblowing thing about the game is scattered messages or morse codes given by the npcs and the actual code of game files contain hints to input linux like command lines in the console of the game and read some encrypted texts. Then you decipher those texts and learn about online game birchkin that if you play it you learn even more secrets about the game. And you have to play some "free to play" games on steam to gather even more clues to learn even more secrets. The rabbit hole goes incredibly deep to finally get a link to an actual youtube video with a possible hint about a true ending or explanation as to what this is about.
Just now i read this is an ARG, an Alternate Reality Game kinda like an html based game i used to play back then, that was called "not pron" or something. All in all this is a good game, i had fun but this whole ARG thing is a little too much for me.\t\t\t\t\t
