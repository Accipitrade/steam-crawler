\r\n\t\t\t\t\tThis game is absolutely worth it. It is a really well executed game that excels in so many ways. Despite the grainy visual style, the game does quite a lot with it and utilises it wonderfully. The humour in the game is phenomenal. It is sure to give you a few laughs. Most impressively of all, what this game does best is actually the horror aspect. Despite being a horror comedy, the game is way scarier than most other horror games because unlike with most horror games, this game will do anything. You never really know what's going to happen next. Maybe there's a jump scare coming up or maybe the whole room is about to change into a massive claw foot bathtub. The sheer lack of predictability with this game, makes it both hilarious and terrifying at the same time. The game's characters, though limited, are also all wonderfully written. It leads the game in the perfect narrative direction for the end. If all that weren't enough, the game is really fun. Each level feels like its own individual thing with its own individual gimmicks and they are all really fun. I would absolutely recommend this game for anyone really.\t\t\t\t\t
