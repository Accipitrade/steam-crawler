\r\n\t\t\t\t\tI don't normally write reviews, but this game has some great potential, and I am really impressed. At first, I was not impressed, but after a few hunts, I began to see how cool this game actually is. There are a large number of entities to identify, and once that is done, you get a chance to exorcise/destroy them. It has elements of Obsidio, Phasmo, and Ghost Hunter's Corp, but nothing is copied, in so far as actual game play. It is it's own unique beast.

There is also a leveling system for your expertise on each ghost type, which is really cool. The more of times you face and exorcise a type of ghost, the more of an expert you become, and you unlock additional information to help in identifying them, like whether or not they can turn lights on and off, or mess with crucifixes on the walls. It is a neat system the other games don't have, plus it adds incentive to keep playing. There is also a class system you unlock at level 50, but I have not reached it yet. Looks pretty cool though, and I look forward to opening my first class.

The game is also 6 player co-op, which is nice, as I usually have 5 or 6 friends looking to play, which usually rules out Phasmo, or Forwarned. The game has also run well in multiplayer so far, with no disconnects or lag.

Now the not so great stuff: the in game chat does not include a radio feature, so if you are not in the immediate proximity, you cannot talk with teammates. Also, the graphix are a bit rough at this point. It will really benefit once the game is more fleshed out. The online community seems to be very small as well, which probably does not help things. I usually play with friends, so I do not care so much, but hopefully, it continues to grow, so the game will continue to develop.

Overall, it is really a cool game with some unique features. It takes several matches to really see where it shines, but once you start getting the hang of it, the game is a blast.

For a game that's been out for a year with so many game breaking bugs is not good.

Example, sometimes only getting 4 notes split between 2 names. Unable to stab the ghost to exercise it when required to do so. Desperately needs events happening for only doors just open and that's all that ever happens. The ghost appears only when to hunt or when caught on a pentagram drawn on the floor by the player. Book says kill with katana, stand around for long periods of time while nothing happens waiting for a hunt just to be killed because swinging the katana did absolutely nothing. Not a single moment is scary or feels unnerving being in any location. The list goes on.

This could be a really solid game if it gets treated well and the dev really focuses on fixing bugs and improving over all quality over quantity.
