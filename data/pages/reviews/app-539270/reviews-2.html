\r\n\t\t\t\t\tWanted a Mahjong solitaire game, and while it does work, it is not well optimized for casual play. Each new puzzle is locked behind beating previous formations, and you cannot choose which puzzles to unlock. Replay value is strong, since each play has a different shuffle, but long load times (given the simplicity of the game) and mandatory leaderboard posting after victories greatly slow down play. In addition, puzzles are frequently unsolvable because of deal (final pairs on top of each other, etc), which means many shuffles are unbeatable from the start of the game. Generally a poor solitaire Mahjong game; would be greatly improved by tweaking the algorithm to limit/eliminate unwinnable deals, to decrease load times, and to allow for disabling the leaderboard in the settings.\t\t\t\t\t

\r\n\t\t\t\t\tA very standard Mahjong game.

There are 70 different stages, but you have to beat a level to unlock the next one so you don't get your choice of layouts until you beat them all. You get a few different tile sets and a few different backgrounds to choose from, which is nice. You also get a few hints if you find yourself stuck, but there is no undo option so if you find yourself in an unwinnable situation you'll have to start the stage over. The game is aesthetically pleasing enough, and the ambient and victory tunes are suitably relaxing. I did find the loading screens to be a little too long for a game that's not particularly demanding. Oh, and you can get all the achievements simply by beating all the stages, which is nice.

This game has been on sale for 90% off, so if you like Mahjong and don't mind that this doesn't do anything new or special with the game, then this will keep you entertained for a few hours.\t\t\t\t\t
