\r\n\t\t\t\t\tA good starter module which includes 7 aircraft, some of which *are not* individually purchasable.

All of these feature a "simplified" startup routine (with no clickable interior), which makes this a good starting point for newbies to the DCS world. The flight dynamics and damage model, however, are still top quality. It also comes with a good starting set of fighters and attackers, which should satisfy the user in the short-term as they get their feet wet.

The downside to this module is that it only comes with training lessons for the Su-27. There are training "lessons" for the F-15C and A-10A, but these are in the form of non-interactable videos.

Interestingly, my lackluster impression of the A-10A led me to purchasing the A-10C module -- which I've found to be a much more enjoyable experience (from an immersion and upgraded avionics suite standpoint). Don't think I'll ever go back to using the A-10A now.

Aside from that, I've only flown the F-15C and Su-27 so far, both of which provided a fun experience when pitting them in BVR engagements and dogfights.\t\t\t\t\t
