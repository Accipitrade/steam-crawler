Starts out very easy and quickly ramps up to a nerve wracking level of information overload. Just when you think you've got everything running smoothly, some small plane you didn't even noticed lands on the runway where you've got a long line of jets crossing, and it's game over.

As some negative reviews have pointed out, this is not anywhere near being a full-blown Air Traffic Control simulator. But that's like arguing that Rock Band sucked because it didn't use real instruments. There's room for more casual, simplified simulations that communicates the frenetic pace of ATC at major airports.\t\t\t\t\t

\r\n\t\t\t\t\tThis is the first time I've actually bought an Airport Madness game, however, I did play the demo of Airport Madness 4 online a lot. It was odd that this version took out the ability to set the traffic level from 1(lowest) to 10(holy crap! Mid-air traffic jams). The only way to set the traffic is in endless mode and even then you only have 3 options. I'd really like to see an update where there is a custom game setting. A option where you pick an airport, set how many planes you want to sucsessfuly land, set the traffic volume and set how many crashes you're allowed before a game over. That's what was missing in this game. I also would not mind if the maps were a little bit larger because it does get a tad crowded, I know that's the point, however, I'd really like more room for delaying flights. So, would I buy this game again? Well, I thought it over and then notived I'd already played it for 85 minutes in one sitting. After realizing that, yes I would buy it again. It's fun and fast paced. The feel and look of the game is great. It goes for the kind of style that is the most fun to the player rather than a realistic radar using controller. I like that because the game would be dull if it was realistic. It's a great little game, but, it is a bit pricey. I waited for the game to be 25% off and that made it worth my money (or lack there-of). I wont give the game a numerical rating since I don't think I could properly explain it. However, I would buy it again even as it is right now. But, I really home Bigfatsimulations considers my ideas.\t\t\t\t\t
